Arsenic (As) mobilization in the Bengal Basin aquifers has been studied for several decades due to the complex redox bio-geochemistry, dynamic hydrogeology and complex nature of dissolved organic matter (DOM). Earlier studies have examined the changes in groundwater As in the dry season before monsoon and during the wet season after monsoonal recharge. To investigate the more immediate influence of recharge during the active monsoon period on As mobilization and DOM character, groundwater samples were analyzed in the pre-monsoon and during the active monsoon period. Groundwater samples were collected from shallow $( < 4 0 ~ \mathrm { m } )$ and deep $( { > } 4 0 \mathrm { m } )$ tube-wells in West Bengal, India. Dissolved $\mathsf { A } s _ { \mathrm { T } }$ in shallow groundwater ranged from 50 to $3 1 5 \mu \mathrm { g } / \mathrm { L }$ exceeding the WHO guideline of $1 0 \mu \mathrm { g / L }$ Shallow groundwater also showed high total dissolved nitrogen, carbon to nitrogen $\left( \mathsf { C } \mathrm { : N } \right) { < } 1$ , and humic-like DOM with a humic:protein ratio ${ > } 1 .$ . By contrast, deep groundwaters contained $\mathsf { A } s _ { \mathrm { T } }$ between 0.5 and $1 1 \mu \mathrm { g / L }$ with carbonaceous and protein-like DOM, $\mathrm { C } { : } \mathrm { N } > 1$ , and humic:protein $< 1$ . Stable isotopes of $\delta ^ { 1 8 } 0$ and $\ S ^ { 2 } \mathrm { H }$ and $\mathrm { C l / B r }$ results indicated three recharge scenarios in the shallow aquifer including direct recharge of dilute rainwater, evaporated surface water, and anthropogenically impacted surface water. Monsoonal recharge did not cause notable changes in $\mathsf { A } s _ { \mathrm { T } }$ in deep or shallow groundwater, including two As hotspots in the Pleistocene aquifer. However, the monsoon did result in a two-fold decrease in ${ \mathsf { S U V } } { \mathsf { A } } _ { 2 5 4 }$ , increase in nitrite and nitrate in the shallow groundwater. The DOM in the deep groundwater at the two As hotspots (with $\mathsf { A s } _ { \mathrm { T } } 1 3 2$ and $7 1 5 \mu \mathrm { g } / \mathrm { L }$ ) had optical properties with much greater humic-like DOM than the surrounding groundwater, which had low $\mathsf { A s } _ { \mathrm { T } }$ and highly protein-like DOM. Overall, these results support that protein-like DOM associated with low groundwater As concentrations and suggest that the monsoonal influence on nitrate and nitrite is limited to shallow aquifers.

# 1. Introduction

Monsoonal precipitation and flooding occurs in South Asia from July to October, bringing over $1 0 0 0 \mathrm { m m }$ of rainfall to the area. For the Bengal Basin, where $9 0 \%$ of the total annual precipitation (1200 to $3 9 4 5 \mathrm { m m }$ ) occurs during the four months (July–October) of wet season (Nath et al., 2008) and $1 5 \%$ of the total precipitation contributes towards the aquifer recharge (Mukherjee et al., 2007a), monsoonal precipitation is a major source of groundwater recharge.

In this hydrologically dynamic landscape, groundwater is the main drinking water source for millions of people. Naturally elevated dissolved arsenic (As) concentrations in groundwater in reducing aquifers of the Bengal Basin have put millions of people at severe health risk. Nearly 20 million people in Bangladesh and over 4 million in West Bengal, India have been estimated to rely on As contaminated $( { > } 1 0 \mu \mathrm { g } / \mathrm { L } )$ groundwater as a major source of drinking water (Chakraborti et al., 2009; Fendorf et al., 2010; Flanagan et al., 2012). Geochemical evidence has indicated that shallow $\scriptstyle ( < 4 0 \mathrm { m } )$ groundwater in the Holocene aquifer composed of younger sediments deposited \~7000 years before present (y BP) has higher dissolved As concentrations (Acharyya et al., 2000; Mukherjee et al., 2007a; Neidhardt et al., 2013). In contrast, the Pleistocene aquifer $2 0 { - } 1 5 0 ~ \mathrm { m }$ deep), which is composed of older sediments deposited $1 2 { , } 3 0 0 { - } 4 8 { , } 6 0 0 \ \mathrm { y }$ BP, contains groundwater with low dissolved As (Acharyya et al., 2000; Moran et al., 2000; Eiche et al., 2008; Burgess et al., 2010; Fendorf et al., 2010; Datta et al., 2011; Hoque et al., 2011; Knappett et al., 2016). The shallow Holocene aquifer in the Bengal Basin is recharged by various sources including precipitation, irrigation return flow and seepage from surface water bodies or losing streams (Ravenscroft et al., 2005).

Rapid monsoonal recharge during the wet season (July–October) in the Bengal Basin (Bangladesh) can elevate the groundwater levels in the shallow aquifer $( 2 0 - 3 0 ~ \mathrm { m } )$ by up to $5 \textrm { m }$ more than in the dry season (November–June) when the monsoon ceases and irrigation pumping resumes (Harvey et al., 2005). Several studies have investigated the influence of monsoonal recharge on dissolved As concentrations in shallow aquifers in the Bengal Basin. A three-year time series study in Bangladesh found no significant seasonal variation in total dissolved arsenic $( \mathsf { A s } _ { \mathrm { T } } )$ concentrations in the shallow aquifer; however a few isolated wells showed a systematic long-term decline in dissolved $\mathsf { A } s _ { \mathrm { T } }$ concentrations (Dhar et al., 2008). Farooq et al. (2010) reported that only $1 4 \%$ of the sampled wells in Murshidabad (West Bengal) showed decrease in post-monsoon dissolved $\mathsf { A } s _ { \mathrm { T } }$ concentrations and attributed this variation to the dilution by As free rain or surface water. Sankar et al. (2014) reported no significant change in dissolved $\mathsf { A } s _ { \mathrm { T } }$ concentrations between pre- and post-monsoon (December–January 2010, 2011 and 2012) samples from Murshidabad (West Bengal). Planer-Friedrich et al. (2012) observed that over 12-months of sampling from the Holocene aquifer in Bangladesh, the dissolved $\mathsf { A } s _ { \mathrm { T } }$ concentrations increased 1.5 times from February (dry period) to August (wet period) and coincided with an increase in the water table. In another study, wet season influences on dissolved As in a shallow aquifer of the Yangtze River Basin in China were reported to have a more complex temporal pattern, showing a temporary decrease in dissolved $\mathsf { A } s _ { \mathrm { T } }$ concentration at the beginning of the precipitation period (May) due to vertical mixing followed by an increase in dissolved $\mathsf { A } s _ { \mathrm { T } }$ during the rest of the wet season (Schaefer et al., 2016). Previous studies collected groundwater samples in the pre-monsoon (dry) and post-monsoon (wet) seasons, but much remains to be understood about the changes underway during the active monsoon period. In the geological setting of the Bengal

Basin, it has been shown that the monsoonal precipitation rapidly recharges the shallow aquifer (Mukherjee et al., 2007b). Therefore, the insights into the effects of recharge on As mobilization during the transition from dry to wet season may advance our understanding of hydrological processes on As mobilization.

Arsenic mobilization in younger Holocene aquifers in the Bengal Basin is known to be driven mainly by microbial reductive dissolution of ferric oxy(hydroxide) rich sediments (McArthur et al., 2001; Dowling, 2002; Hasan et al., 2007). Labile organic matter originating from aquifer sediments (McArthur et al., 2004; Sengupta et al., 2008; Datta et al., 2011; Neumann et al., 2014) or drawn to the depth from surface water sources (Harvey et al., 2006; Neumann et al., 2010; Lawson et al., 2016) serves as the energy source for microbial reductive dissolution. Additionally, biologically recalcitrant dissolved organic matter (DOM), such as fulvic acid, has been shown to enhance As mobilization by serving as an intermediate electron shuttle (Kulkarni et al., 2018), forming aqueous complexes with Fe and As (Sharma et al., 2010; Liu et al., 2011), and competing with As for sorption sites (Bauer and Blodau, 2006). Recent studies have documented the predominantly humic-like character of DOM in shallow high As groundwater in Bangladesh (Mladenov et al., 2010, 2015) and West Bengal, India (Kulkarni et al., 2017; Vega et al., 2017). However, less is known about the influence of aquifer recharge during monsoon on the quality of groundwater DOM as it relates to As mobilization.

Previous efforts to track temporal changes in dissolved organic carbon (DOC) concentration in groundwater have also shown contrasting effects of monsoonal recharge. Majumder et al. (2016) observed that the DOC concentration increased during the post-monsoon season in Nadia district in West Bengal (India) and was related to increased reduction of As (V) to As (III). In contrast, Planer-Friedrich et al. (2012) reported an inverse seasonal trend between DOC and dissolved $\mathsf { A s } _ { \mathrm { T } } ,$ as the lowest DOC concentration coincided with the highest $\mathsf { A } s _ { \mathrm { T } }$ concentration. However, such seasonal changes in the DOM properties have not been previously investigated, especially during the active monsoon period.

In terms of spatial variability in DOM chemical quality, the deeper groundwater in Pleistocene aquifers that is generally considered to be free from As contains substantially low DOC concentrations (Sutton et al., 2009; Datta et al., 2011; Sankar et al., 2014) that limit the release of As from these sediments (Harvey et al., 2002; Postma et al., 2007; Berg et al., 2008; Polizzotto et al., 2008; Fendorf et al., 2010). Heterogeneous occurrences of elevated As levels in deeper groundwater in the Pleistocene aquifer have been reported, although not very commonly (Burgess et al., 2010; Erban et al., 2013) and often are attributed to drawdown of As rich shallow groundwater to greater depths (Neidhardt et al., 2013). It has been suggested that the presence of discontinuous clay lenses with higher organic carbon content within the deep aquifer providing labile DOM for microbial metabolism (Mukherjee et al., 2007a, 2007b; Schaefer et al., 2016) may be responsible for such heterogeneous elevated As levels. However, the chemical quality of DOM associated with As hotspots in the Pleistocene aquifer still has not been investigated.

In this study, we collected and analyzed shallow (high As, Holocene aquifer, Nadia district) and deep (low As, Pleistocene aquifer, Hooghly district) groundwater samples from drinking water tube wells. Each tube well was sampled once during the dry season (May–June 2015) and once during the monsoon (July–August 2015). Analyses of groundwater chemistry and optical properties of DOM were conducted in shallow and deep groundwater samples to investigate the influence of monsoonal recharge on As biogeochemistry in the Bengal Basin.

# 2. Materials and methods

# 2.1. Study site

Groundwater samples were collected from 11 existing drinking water hand-pumped tube-wells in village Shahispur $( 2 3 ^ { \circ } 4 ^ { \prime } 1 8 . 4 7 ^ { \prime \prime } \mathrm { N } ,$ $8 8 ^ { \circ } 3 6 ^ { \prime } 3 4 . 9 6 ^ { \prime \prime } \mathrm { E }$ , ${ \sim } 1 0 ~ \mathrm { m }$ above sea level) and 10 tube-wells in village Chakudanga $( 2 3 ^ { \circ } 4 ^ { \prime } 1 8 . 4 7 ^ { \prime \prime } \mathrm { N }$ , $8 8 ^ { \circ } 3 6 ^ { \prime } 3 4 . 9 6 ^ { \prime \prime } \mathrm { E }$ , $\mathrm { \sim } 1 0 \mathrm { m }$ above sea level) in Nadia district (Fig. 1). These two sites are $\mathrm { \sim } 3 \mathrm { k m }$ away from each other and ${ \sim } 1 2 \mathrm { k m }$ east of river Hooghly. Tube-wells $\mathrm { i n } = 5$ , from each village) in villages Bele $( 2 3 ^ { \circ } 3 ^ { \prime } 4 0 . 5 5 ^ { \prime \prime } \mathrm { N }$ , $8 8 ^ { \circ } 2 1 ^ { \prime } 7 . 8 1 ^ { \prime \prime } \mathrm { E }$ ) and Radhanagar $2 3 ^ { \circ } 3 ^ { \prime }$ $1 8 . 2 2 ^ { \prime \prime } \mathrm { N }$ , $8 8 ^ { \circ } 2 0 ^ { \prime } 4 1 . 1 0 ^ { \prime \prime } \mathrm { E }$ ) in Hooghly district (Fig. 1) were sampled. These two sites are $\mathrm { \sim } 1 \ \mathrm { k m }$ away from each other and ${ \sim } 1 2 \ \mathrm { k m }$ west of river Hooghly. Samples from Nadia district represent typically shallow high dissolved $\mathsf { A } s _ { \mathrm { T } }$ aquifer, while those from Hooghly district represent typically deeper low dissolved $\mathsf { A } s _ { \mathrm { T } }$ aquifers. In addition, four preinstalled piezometers (Biswas et al., 2011) at Chakudanga and Shahispur each were also sampled. The samples were collected twice from the same tube-well, once during May–June 2015, which are considered as pre-monsoon samples and once after the monsoon started (July–August 2015) which are considered as during monsoon samples. This classification was done based on continuous water level data presented in Harvey et al. (2005) for Bangladesh, Biswas et al. (2014) for Nadia (West Bengal), while the actual monsoon started on 15th June 2015 in the study area. During the year 2009–2010, Biswas et al.

(2014) reported water table at Sahispur to fluctuate from $5 \mathrm { m }$ to $1 \textrm { m }$ (bgl), and at Chakudanga from $4 . 5 ~ \mathrm { m }$ to $1 \textrm { m }$ (bgl) in May–June and August–September respectively. Surface water samples from ponds and streams were collected during July–August only.

Sediment core analyses at Shahispur and Chakudanga have shown that the aquifer is composed of the dark gray colored As rich sand (1.8 to $1 2 . 6 \mathrm { m g / k g ) }$ that is overlain by a silty clay layer of overbank deposits forming the surface aquitard (Biswas et al., 2014). The surface aquitard in Sahispur extended down to $3 . 3 5 \mathrm { ~ m ~ }$ (bgl) and $3 . 2 0 \textrm { m }$ (bgl) at Chakudanga. A continuous aquifer extended down to the depth of $3 9 . 2 \mathrm { ~ m ~ }$ (Sahispur) and of $4 5 . 5 ~ \mathrm { ~ m ~ }$ (Chakudanga) was observed (Neidhardt et al., 2013). Total organic carbon (TOC) up to $0 . 3 \%$ by weight was observed in top clayey layers (up to $4 \mathrm { m } \mathrm { b g } \mathrm { l }$ ) and ${ \sim } 0 . 1 \%$ in extended sand aquifer, however, TOC content peaked up to $0 . 5 \%$ at the depth of $\mathrm { \sim } 3 9 \mathrm { \ m }$ at Sahispur site where a clay layer was encountered at the depth of $3 9 . 2 \mathrm { m }$ (Biswas et al., 2014). The organic matter in southern Nadia district $( 2 3 ^ { \circ } 0 4 ^ { \prime } 5 5 ^ { \prime \prime } \mathrm { N } / 8 8 ^ { \circ } 3 0 ^ { \prime } 4 4 ^ { \prime \prime } \mathrm { E } )$ has been characterized by the presence of abundant high molecular weight n-alkanes, hopanes and steranes and less abundant n-alkanols and sterols (Rowland et al., 2006). At both the sites, similar concentrations of iron mineral $\mathrm { ( F e _ { 2 } O _ { 3 } , }$ $3 \mathrm { - } 4 \%$ by weight) and sedimentary As $( 3 - 5 \mathrm { m g } / \mathrm { k g } )$ were observed.

A typical lithology in Hooghly district (Radhanagar and Bele sites) has been described by Kundu and Mandal (2009) and characterized by fine light gray sand $( 6 7 - 9 2 \mathrm { m } )$ and medium sand with small gravels (92–137 m). These sediments can be considered to be of similar age as the sediments in Murshidabad district ${ \widetilde { \mathbf { \Lambda } } } { \widetilde { \mathbf { \Lambda } } } ^ { * 1 0 0 1 } { \mathrm { k m } }$ north of Bele and Radhanagar sites, on west side of river Hooghly, low As sites) which were characterized as Pleistocene sediments that may be deposited 12,300–48,600 y BP (Sankar et al., 2014). Hooghly district in general $\ l ( \mathrm { n } ~ = ~ 2 2 1 2$ ) has been reported to have low As groundwater (Chakraborti et al., 2013). However, the detailed As geochemistry at Bele and Radhanagar has not been studied earlier. Sankar (2013) and Sankar et al. (2014) have reported sediment characteristics of Pleistocene sediments in Murshidabad district with yellow-reddish color sands with $5 { - } 1 0 ~ \mathrm { m g / k g }$ of sedimentary As and $0 . 0 7 \mathrm { - } 0 . 1 \%$ by weight TOC. Ghosh et al. (2015) described the organic matter characteristics of Pleistocene sediments from another site in Nadia district $( \sim 1 2 \mathrm { k m }$ south-west of Sahispur site) to contain high molecular weight (HMW) n-alkanes, low molecular weight (LMW) n-alkanes, mono-saturated n-alkanoic acids, LMW n-alkanoic acids, sterol and stanol monomers and lipids. The selected sites for this study are within $2 5 \mathrm { k m }$ distance and receive the similar average annual precipitation (1200 to $3 9 4 5 \ \mathrm { m m }$ during June/July–September/October (Majumder et al., 2016).

# 2.2. Sample collection and storage

The drinking water tube-wells were pumped for at least 1 h continuously before collecting the samples. All the samples were filtered using pre-combusted (4 h, $4 5 0 ^ { \circ } \mathrm { C }$ ) and pre-rinsed (with $3 0 0 ~ \mathrm { { m L } }$ of sample water) $0 . 7 \mu \mathrm { m }$ glass fiber filter (GFF) using a handheld vacuum pump in the field. Samples were collected in acid-cleaned HDPE bottles that were rinsed with the filtered sample water three times. For each tubewell, three samples were collected, 1) unacidified (for analysis of anions and DOM), 2) acidified with $0 . 2 \%$ v/v reagent grade $\mathrm { H N O } _ { 3 }$ (for $\mathsf { A s } _ { \mathrm { T } }$ analysis) and 3) acidified with $0 . 2 \%$ v/v reagent grade HCl (for analysis of total organic carbon and nitrogen). Sample collection during the monsoon was performed under a temporary shelter to avoid mixing with rain. All samples were transported in coolers and stored at $4 ^ { \circ } C$ in dark.

# 2.3. Analyses

Total dissolved As $( \mathsf { A s } _ { \mathrm { T } } )$ concentrations were measured using ICPMS (Dept. of Public Health, SDSU). Inorganic anions $( \mathbf { C l } ^ { - }$ , Br−, $\mathrm { F ^ { - } }$ , ${ \mathsf { N O } } _ { 3 } ^ { - }$ , $\mathsf { N O } _ { 2 } ^ { - }$ , $\mathsf { P O } _ { 4 } ^ { 3 - }$ , $S 0 _ { 4 } ^ { 2 - }$ , ${ \mathsf { A } } s ^ { 5 + }$ ) were measured using Dionex Ion Chromatography System (Dept. of Biology, SDSU). It is to be noted that unacidified samples were used for IC analysis and therefore results of ${ \mathsf { A } } s ^ { 5 + }$ analysis should be interpreted with caution because the samples were not preserved to maintain As speciation during transport and storage period. Stable isotopic composition ( $\delta ^ { 2 } \mathrm { H }$ and $\delta ^ { 1 8 } 0$ ) in filtered unacidified $\mathrm { \Delta } \ln = 1 0$ ) samples were determined by a Thermo Finnigan Delta Plus mass spectrometer with dual inlet and continuous flow capabilities (Dept. of Biology, KSU) and were standardized to the Vienna Standard Mean Ocean Water (VSMOW) reference material. Nonpurgable organic carbon (NPOC or DOC) and TDN were measured by thermic oxidation using Shimadzu TOC/TN Analyzer (Dept. of CCEE, SDSU) in filtered samples acidified with HCl. Detailed procedure to analyze DOM in groundwater is described in Kulkarni et al. (2017). Briefly, absorbance and 3-dimensional fluorescence spectra were acquired using Horiba Aqualog benchtop fluorometer (Dept. of CCEE, SDSU), with excitation wavelength range of $2 4 0 \mathrm { n m }$ to $4 5 0 \mathrm { n m }$ with increment of $3 \mathrm { n m }$ and emission wavelengths of $3 0 0 \mathrm { n m }$ to $6 0 0 \mathrm { n m }$ . Absorbance at $2 5 4 \mathrm { n m }$ intensity was used to calculate specific ultra-violet absorbance $\left( { \mathsf { S U V A } } _ { 2 5 4 } \right)$ which indicates aromaticity of DOM (Weishaar et al., 2003) when normalized with DOC concentration. Fluorescence spectra were corrected for inner filter effect, Rayleigh scattering and normalized using Raman area of blank $1 8 . 3 \ : \mathrm { M } \Omega \cdot \mathrm { c m }$ Mili-Q ultrapure water). Different fluorescence peak intensities and fluorescence indices as previously summarized in Kulkarni et al. (2017) were calculated using corrected 3D excitation emission matrices (EEMs). Further, a three component PARAFAC model was validated using 78 corrected EEMs. Model validation was performed by split half validation and random initialization techniques (Stedmon and Bro, 2008). Statistical analyses (paired two-tailed t-test with unequal variances, linear regression) were performed on the data to identify significant differences and correlations.

# 3. Results and discussions

# 3.1. Spatial distribution of As and DOM

Drinking water tube wells tapping Holocene sediments (Chakudanga and Shahispur, $\mathrm { n } = 2 0$ , depth range 15 to $4 5 ~ \mathrm { m }$ ) were found to have average dissolved $\mathsf { A } s _ { \mathrm { T } }$ concentrations ( $1 3 8 \mu \mathrm { g / L }$ in premonsoon and $1 4 3 \mu \mathrm { g / L }$ during monsoon), which exceeded the World Health Organization (WHO) safe limits of $1 0 \mu \mathrm { g / L }$ (Fig. 2). In contrast, the average dissolved $\mathsf { A } s _ { \mathrm { T } }$ concentrations in samples ${ \mathrm { ~ \ : ~ } } n = 8$ ) from the tube wells tapping Pleistocene sediments (Radhanagar and Bele sites; average of $2 \mu \mathrm { g } / \mathrm { L }$ during both pre-monsoon and during monsoon for a depth range of 60 to $1 0 9 \mathrm { m }$ ) were below national and WHO safe limits (Fig. 2). However, the two tube wells at Bele (Pleistocene site) showed exceptionally high dissolved $\mathsf { A } s _ { \mathrm { T } }$ concentrations (132 and $7 1 5 \mu \mathrm { g } / \mathrm { L }$ ) respectively (Fig. 2), which is discussed separately. The overall spatial and vertical As distribution is in agreement with previous studies that reported elevated As levels in shallow aquifers tapping the Holocene aquifer in the Bengal Basin (Fendorf et al., 2010).

Our data showed that the high dissolved $\mathsf { A } s _ { \mathrm { T } }$ groundwater was associated with lower sulfate $( \mathsf { S O } _ { 4 } ^ { 2 - } )$ concentrations. Several earlier studies have demonstrated that sulfate concentrations tend to be low in high As groundwater and high in low As groundwater (Harvey et al., 2002; Polizzotto et al., 2005; Jessen et al., 2008; Buschmann and Berg, 2009; Kar et al., 2010; Bhowmick et al., 2013; Sankar et al., 2014; Vega et al., 2017). Previous laboratory and field studies have shown that $S 0 _ { 4 } ^ { 2 - }$ could serve as electron acceptor for sulfate reducing bacteria and may limit As mobilization by the formation of Fe-As-S minerals (Rittle et al., 1995; Jong and Parry, 2003; Kirk et al., 2004, 2010; Keimowitz et al., 2007; Saunders et al., 2008; Omoregie et al., 2013). Although the difference between sulfate concentrations in high and low As sites were found to be statistically significant $\mathrm { p } < 0 . 0 5$ , Table S5), the magnitude of the difference was small; the $S 0 _ { 4 } ^ { 2 - }$ concentration at high As sites ranged from 0 to $4 \mathrm { m g / L }$ , while at low As sites it ranged from 2 to $8 \mathrm { m g / L }$ (Tables S1 and S2). In addition, the correlation between As and sulfate was low $\mathbf { R } ^ { 2 } = 0 . 3$ ). Therefore, while the distribution of As and sulfate in our study appears to be consistent with the theoretical understanding of As-sulfate interactions in reducing aquifers, sulfate may not be the only control on As mobility at studied sites.

We found that DOC concentrations were also positively and significantly correlated (Table S5) with dissolved $\mathsf { A } s _ { \mathrm { T } }$ $\mathrm { R } ^ { 2 } = 0 . 3$ , $\mathsf { p } < 0 . 0 0 5$ ). Significant relationship between dissolved $\mathsf { A } s _ { \mathrm { T } }$ and DOC has been demonstrated in other studies (McArthur et al., 2001; Dowling, 2002; Hasan et al., 2007). The Holocene sites east of river Hooghly had higher DOC than the Pleistocene sites west of river Hooghly (Tables S1 and S2). The pre-monsoon DOC concentrations ranged from 0.9 to $2 . 0 \ : \mathrm { m g / L }$ at high As Holocene sites and 0.5 to $1 \ \mathrm { m g / L }$ at low As Pleistocene sites (Fig. 3a). There was also a significant positive correlation between dissolved $\mathsf { A } s _ { \mathrm { T } }$ and total dissolved nitrogen (TDN) concentrations (Fig. 3b) during the pre-monsoon $\mathrm { R } ^ { 2 } = 0 . 2 , \mathrm { p } = 0 . 0 0 2 ,$ ) season (Table S5). An average of $2 \mathrm { m g / L }$ and $0 . 6 \mathrm { m g / L }$ TDN was found in high and low As tubewells, respectively, during the pre-monsoon season. At a similar site with shallow groundwater in Araihazar, Bangladesh, Mladenov et al. (2015) reported TDN concentrations in a similar range for high As groundwater. Our analysis also showed that during pre-monsoon, $8 7 \%$ and $8 5 \%$ of TDN at high and low As sites, respectively, was in the reduced form of N $\mathrm { N H } _ { 4 } ^ { + }$ + Organic N) (Fig. 5). This is consistent with several studies that highlighted the role of dissolved nitrogen dynamics in As mobilization. For example, field observations of a concurrent presence of $\mathrm { N H } _ { 4 } ^ { + }$ in high As groundwater (Nickson et al., 1998; McArthur et al., 2001, 2004; Harvey et al., 2002; Smedley and Kinniburgh, 2002; Berg et al., 2008; Nath et al., 2008) provided evidence of microbial oxidation of DOM containing organic nitrogen. Results from our study further showed that in high As $_ { \cdot > 5 0 }$ $\mu \mathrm { g } / \mathrm { L }$ ) samples, the molar ratio of carbon to nitrogen (C:N) was $^ { < 1 }$ (Fig. 3c). This observation supports the findings of batch incubation experiments (Solaiman et al., 2009), which showed that As mobilization was tightly regulated by the C:N ratio due to the N requirements of Fe reducing microorganisms. Solaiman et al. (2009) also found that higher As mobilization coincided with a lower C:N ratio under reducing conditions.

# 3.2. Spatial changes in DOM chemical and optical characteristics

In addition to the groundwater quality discussed above, which had statistically significant correlations with dissolved $\mathsf { A } s _ { \mathrm { T } }$ (Table S5), the intensity of UV absorbance at $2 5 4 \mathrm { n m }$ $ { \mathrm { \Delta A b s } } _ { 2 5 4 } )$ was also positively correlated with dissolved $\mathsf { A } s _ { \mathrm { T } }$ concentrations during both pre-monsoon and monsoon $\mathrm { R } ^ { 2 } = 0 . 3$ , $\mathsf { p } < 0 . 0 0 5$ ). Further, $\mathsf { S U V } \mathsf { A } _ { 2 5 4 }$ calculated as Abs254 intensity normalized with DOC concentration, was also weakly correlated with dissolved $\mathsf { A } s _ { \mathrm { T } }$ $\mathrm { R } ^ { 2 } = 0 . 2 , \mathrm { p } = 0 . 0 1 2$ ) during monsoon. Higher ${ \tt A b s } _ { 2 5 4 }$ intensity in high As sites (Fig. 3d) is consistent with higher DOC concentrations and suggests generally higher aromaticity (Ohno, 2002). However, in this study, $\mathtt { A b s } _ { 2 5 4 }$ and $\mathsf { S U V } \mathsf { A } _ { 2 5 4 }$ were not utilized for the characterization of DOM due to potential interference by Fe. $\mathtt { A b } s _ { 2 5 4 }$ can be influenced by Fe (III) concentration, as Fe (III) absorbs significantly near $2 5 4 ~ \mathrm { { n m } }$ (Weishaar et al., 2003). Although Fe (III) species have been shown to be low at this site (Biswas et al., 2011, 2014; Neidhardt et al., 2013; Majumder et al., 2016), during UV measurement, Fe (II) may oxidize quickly and result in increased absorbance (Poulin et al., 2014). Therefore, $\mathtt { A b s } _ { 2 5 4 }$ and $\mathsf { S U V } \mathsf { A } _ { 2 5 4 }$ values do not necessarily represent the higher aromaticity, but may also reflect higher Fe concentrations in samples with high As concentrations.

The fluorescence parameters also revealed differences between low and high As sites. Lower freshness index $( \beta ; \alpha )$ in high As sites (0.77) and higher $\beta { : } \alpha$ (Fig. 3e) in low As sites (0.98) represents biologically more- and less-processed DOM (Fellman et al., 2010), respectively, and is consistent with previously reported values in the Bengal Basin (Kulkarni et al., 2017; Vega et al., 2017). Similarly, the humification index (HIX) indicating the extent of humification of DOM (Ohno, 2002; Zsolnay, 2003) was also correlated (Fig. 3f) with $\mathsf { A } s _ { \mathrm { T } }$ concentration ( $\mathrm { R } ^ { 2 } = 0 . 2$ , ${ \mathsf { p } } = 0 . 0 1 3$ ). Average HIX values of 6 and 1 in high and low As sites, respectively, were observed in this study. Similar patterns have been shown previously by Kulkarni et al. (2017) in the Murshidabad district, where HIX values of 7 and 3 were measured in high and low As sites, respectively. In wells that also contained elevated manganese concentrations, HIX values of 11 and 5 were present in high and low As sites, respectively, Vega et al. (2017) reported. Fluorescence spectra used for DOM characterization were further resolved using a 3- component PARAFAC model (Fig. 4), which identified components as terrestrial humic-like (C1), humic-like and impacted by agriculture or wastewater activity (C2), and microbial (tryptophan or tyrosine) protein-like (C3). Analysis of individual component intensities showed that components C1 $\mathrm { R } ^ { 2 } = 0 . 1$ , $\mathsf { p } = 0 . 0 3 / \mathsf { R } ^ { 2 } = 0 . 3$ , $\mathsf { p } < 0 . 0 0 5$ ) and C2 $\mathrm { R } ^ { 2 } = 0 . 1$ , $\mathsf { p } = 0 . 0 2 / \mathsf { R } ^ { 2 } = 0 . 4$ , $\begin{array} { r }  \mathsf { p } < 0 . 0 0 5 \ \end{array}$ ) were positively correlated with $\mathsf { A } s _ { \mathrm { T } }$ concentrations during pre-monsoon and monsoon, while C3, the protein-like component was not significantly correlated with $\mathsf { A s } _ { \mathrm { T } }$ Several other studies have now shown that low protein-like fluorescence is typical of high As groundwater environments (Mladenov et al., 2010; Kulkarni et al., 2017; Vega et al., 2017). Based on three PARAFAC components in this study, humic:protein ratio (Kulkarni et al., 2017) was calculated (Tables S1 and S2) and was found to be weakly correlated (Table S5) with dissolved $\mathsf { A } s _ { \mathrm { T } }$ during pre-monsoon $\mathrm { R } ^ { 2 } = 0 . 2$ , ${ \mathsf { p } } = 0 . 0 1 2$ ) and even more significantly during monsoon $( \mathtt { R } ^ { 2 } = 0 . 4 , \mathtt { p } < 0 . 0 0 5 )$ . This observation was also consistent with previous observations in the Murshidabad district (West Bengal), that the humic: protein ratio was higher in groundwater with high dissolved $\mathsf { A } \mathsf { s } _ { \mathrm { T } }$ (Kulkarni et al., 2017; Vega et al., 2017). However, one of the four PARAFAC components identified ( $. \sim 2 5 \%$ of total fluorescence) as microbial humic-like $\mathrm { E x } 2 4 0 \ \mathrm { n m / E m } 3 8 8 \ \mathrm { n m } )$ in deep low $\mathsf { A } s _ { \mathrm { T } }$ groundwater of Murshidabad (Kulkarni et al., 2017; Vega et al., 2017) was not identified in this study. While the exact molecular structure of the compound resulting in this PARAFAC component is unknown, several possibilities can be considered. Lower microbial populations in Hooghly district compared to in Murshidabad could be responsible for the absence of this microbial humic-like component in this study. If the component is related to surface derived microbial inputs, then different land use patterns and surface-groundwater interactions could have an influence on the variation in fluorescent components. Further investigation including characterization of molecular structure of DOM and microbial population, is required to fully understand the spatial distribution of fluorescent DOM components.

Analysis of C:N molar ratio and humic:protein ratio revealed that $8 6 \%$ of all samples from high As sites had $\mathrm { C } { : } \mathrm { N } < 1$ (i.e. ${ \mathsf C } < { \mathsf N }$ ) and humic:protein ${ > } 1$ (i.e. humic-like $>$ protein-like) (Fig. 5). The presence of humic-like DOM in high As groundwater results from both the presence of lignaceous (plant or soil-derived) organic matter and the microbial consumption of protein-like DOM. The degradation of protein-like DOM, acting as an electron donor, would lead to As mobilization, enrichment of humic-like DOM, and a lower C:N ratio for DOM in the groundwater. The question is when such microbial degradation of protein-like

DOM would have occurred. Previous studies using radiocarbon dating indicate that reactive DOM in aquifers in South Asia is hundreds of years old (Simone, 2010; Mailloux et al., 2013) and suggest that the presence of humic-like DOM is not a recent occurrence. Instead, it has been suggested that the presence of humic DOM with its low C:N ratio is related to either hundreds of year old microbial processing of labile DOM (Mailloux et al., 2013) or concurrent deposition of humic organic matter and sediments in the Holocene (Mladenov et al., 2015). Evidence from our study further reveals that groundwater containing labile electron donor (protein-like DOM) rather than humic DOM does not have low C:N, despite having sufficient electron acceptor (Fe (oxy)hydroxides) and presumably ample microbial communities.

Another important consideration in understanding the influence of humic DOM on arsenic mobilization is the electron shuttling role that has now been demonstrated for fulvic acids isolated from similar reducing aquifers (Chen et al., 2017; Kulkarni et al., 2018). For example, Bangladesh fulvic acids showed higher electron accepting capacity than Suwanee River Fulvic Acid (Kulkarni et al., 2018), which may lead to the acceleration of As mobilization. In an incubation experiment by Solaiman et al. (2009) where organic matter with different C:N ratios was added to the As rich ferric oxyhydroxides sands, the As release from sediments was greatest for the organic matter with the lowest C:N ratio. In this study, the high As groundwater contains more nitrogenous (C:N $^ { < 1 }$ ) and also more humic-like (humic:protein ${ > } 1$ ) DOM (Fig. 5). Therefore, it appears that, in addition to the nitrogenous nature of DOM, the more humic-like character of DOM regulates As mobilization, possibly via mechanisms like electron shuttling.

# 3.3. Monsoonal influence on groundwater chemistry

Monsoonal recharge had an influence on both the groundwater levels in piezometers and on groundwater quality. Groundwater levels were observed in piezometers at Holocene sites (Shahispur and Chakudanga) to be elevated by ${ \sim } 2 . 9 \mathrm { m }$ at both sites during monsoonal recharge (July–August). Biswas et al. (2014) has previously documented with 20 months monitoring data for these piezometers that the groundwater table rises by $\sim 3 \textrm { m }$ at Shahispur and $\sim 2 \textrm { m }$ a t Chakudanga during the monsoon season. Stable isotope values also reflect the influence of monsoon on groundwater recharge. The stable isotopes ( $\mathrm { \Phi ^ { \delta 1 8 } 0 }$ and $\delta ^ { 2 } \mathrm { H }$ ) at the study sites plot in between the local meteoric water line (LMWL) and pond evaporation line (Fig. 6a). Two samples from the Shahispur site showed slightly more enriched $\delta ^ { 1 8 } 0$ values $- 3 . 5$ and $- 2 . 9 \text{‰}$ ) than samples from Chakudanga ( $- 4 . 4$ and $- 3 . 8 \mathrm { ‰ }$ ) (Fig. 6b). Two of four wells in our study showed depletion in $\boldsymbol { \delta } ^ { 1 8 } \boldsymbol { 0 }$ during the monsoon (Fig. 6), which agrees with findings of Biswas et al. (2014) where a slight depletion in $\boldsymbol { \delta } ^ { 1 8 } \boldsymbol { 0 }$ during monsoon and enrichment during the dry period suggests major recharge by monsoonal precipitation. Similar observations were made in previous studies suggesting that the shallow aquifer in western Bengal Basin, is recharged mostly through monsoonal precipitation, which usually has a depleted $\boldsymbol { \delta } ^ { 1 8 } \boldsymbol { 0 }$ $( - 4 . 8 \text{‰}$ signature, while the recharge from partly evaporated surface waters cannot be ruled out, as indicated by enriched $\delta ^ { 1 8 } \bar { 0 }$ values (Zheng et al., 2005; Mukherjee et al., $2 0 0 7 \mathrm { a }$ ; McArthur et al., 2012; Biswas et al., 2014).

Analyses of the Cl/Br mass ratio (Fig. 7) revealed that the groundwater samples from Chakudanga (low As site) may have been recharged with dilute rainwater as indicated by $\mathrm { C l / B r } < 1 0 0$ and $\mathrm { C l } ^ { - } < 1 0 \mathrm { m g / L }$ . It has been shown that $\mathbb { C } 1 ^ { - }$ concentrations ${ < } 1 0 \ \mathrm { m g / L }$ in groundwater can be derived from rainwater (McArthur et al., 2012). Increased $\mathtt { C l } ^ { - }$ concentration in Radhanagar and Bele (low As, deeper) samples suggest that the groundwater received evaporated recharge. However, the lower $\mathrm { C l / B r }$ ratio in these samples suggest minimal anthropogenic contamination as they plot below the mixing lines. The possible sources of contamination in our study area may be due to recharge from village ponds and domestic wastewater.

The low Cl/Br of $< 2 0$ and $\mathrm { C l } ^ { - } { < } 1 0 \mathrm { m g / L }$ in samples from Chakudanga that plotted below dilute recharge zone (Fig. 7) may be attributed to recharge with dilute and uncontaminated rainwater or surface water or to the decomposition of organic matter. Previous studies have shown that the decomposition of natural organic matter increases bromide over chloride. For example, the $\mathrm { C l / B r }$ of animal dung leachate was found to be 35–167 (Hudak, 2003) and $\mathrm { C l / B r }$ of 4 was found to result from the degradation of lignite and peat (Nissenbaum and Magaritz, 1991). Values between 6 and 10 have been reported for the degradation of soil organic matter (Gerritse and George, 1988) and peats (Biester et al., 2005, 2012). In contrast, samples that plotted above the seawater mixing line and have $\mathtt { C l } ^ { - }$ concentration ${ > } 1 0 \mathrm { m g / L }$ likely received the recharge from evaporated and contaminated sources.

Further, dissolution of evaporitic minerals may contribute to the $\mathrm { C l } /$ Br ratios of some samples. For example, the samples from Shahispur (high As site, shallow aquifer) collected during the monsoon (July–August) plotted very close to the salt mixing line, which indicates the dissolution of evaporitic minerals along the flow paths (Knappett et al., 2016). Previous work in our study area (Majumder et al., 2016) also reported that groundwater in the floodplain with Cl/Br molar ratio of 60–100 was influenced by leachate and halite dissolution. In our study, few other pre-monsoon samples plotted above the salt mixing line. It should be noted that these samples had bromide concentrations below the detection limit $0 . 0 1 \mathrm { m g / L }$ ) and their true position on the plot may be above from where they are currently plotted (Fig. 7).

Additionally, the chloride values varied by $\pm 2 5 \mathrm { m g / L }$ in the samples collected in pre-monsoon and during monsoon seasons from shallow $( < 4 0 \mathrm { m } )$ wells compared to the variation of $\pm 0 . 2 \mathrm { m g / L }$ in deeper wells tapping the Pleistocene aquifer (Fig. S1). Such a higher variation in $\mathtt { C l } ^ { - }$ concentrations in shallow groundwater can be explained by two different phenomena. Given that $\mathbb { C } 1 ^ { - }$ is a conservative tracer, the decrease in $\mathtt { C l } ^ { - }$ concentrations reflects infiltration of low $\mathbb { C } 1 ^ { - }$ rainwater that dilutes the groundwater. On the contrary, increases in $\mathtt { C l } ^ { - }$ concentrations represent the interaction of shallow groundwater with anthropogenically impacted surface water sources such as in the case of Shahispur samples, which plotted above the salt mixing line and near the sewage mixing line (Fig. 7). Similar effects of recharge on $\mathtt { C l } ^ { - }$ have been previously seen in several other studies (Ravenscroft et al., 2005; Sankar et al., 2014; Majumder et al., 2016) in the Bengal Basin.

We further observed increases in dissolved $\mathsf { A } s _ { \mathrm { T } }$ as high as $1 4 5 \mu \mathrm { g / L }$ and decreases as low as $5 2 \mu \mathrm { g } / \mathrm { L }$ during monsoon. Overall, the variability in dissolved $\mathsf { A } s _ { \mathrm { T } }$ concentrations was highest in shallow groundwater. The higher variability of dissolved $\mathsf { A } s _ { \mathrm { T } }$ concentrations in shallow groundwater than in deep groundwater (Fig. 2) may also reflect the greater reactivity of As in the reducing Holocene aquifer, where redox reactions can result in As precipitation and dissolution. Also, while the changes in individual tube-wells in our study are considerable, we are not able to point to either a consistent increase or decrease in dissolved $\mathsf { A } s _ { \mathrm { T } }$ concentrations in the studied aquifer, as there was only a $4 \%$ increase in average dissolved $\mathsf { A } s _ { \mathrm { T } }$ concentrations between samples collected during the pre-monsoon and monsoon seasons (Table S6). In similar floodplain of the Hooghly River in Nadia district, Majumder et al. (2016) also reported a subtle but statistically insignificant $\cdot \mathtt { p } >$ 0.05) increase of $7 \%$ in dissolved $\mathsf { A } s _ { \mathrm { T } }$ concentrations between premonsoon (April–May) and post-monsoon (December) samples. In our study, the speciation of As did not change substantially during the monsoon and remained primarily as the reduced form as $9 9 \%$ of $\mathsf { A } s _ { \mathrm { T } }$ is As (III). Similarly, Biswas et al. (2014) found no change in As speciation during monsoon but an increase in $\mathsf { A } s _ { \mathrm { T } }$ in post-monsoon season.

Our shallow groundwater samples showed statistically significant (p $< 0 . 0 5 $ ) three-fold increase in nitrite $( \mathsf { N O } _ { 2 } ^ { - } )$ concentrations during monsoonal recharge. During the pre-monsoon season, $8 7 \%$ of the TDN was in the form of $( \mathrm { N H _ { 4 } ^ { + } + 0 }$ rganic N), which decreased to $7 0 \%$ during monsoonal recharge (Fig. 8). Increase in $\mathsf { N O } _ { 2 } ^ { - }$ concentrations (Table S6) may be attributed to the intermediate step in nitrification of $\mathrm { N H } _ { 4 } ^ { + }$ under gradually oxidizing conditions during monsoonal recharge (McClain et al., 1994). The presence of $\mathrm { N H } _ { 4 } ^ { + }$ in groundwater of Bengal Basin has always been linked to the degradation of nitrogenous organic matter (Nickson et al., 1998; McArthur et al., 2001, 2004; Harvey et al., 2002; Smedley and Kinniburgh, 2002; Berg et al., 2008; Nath et al., 2008), and our pre-monsoon samples agree with that observation. The change in the redox state of the aquifer from highly reducing to oxidizing during the rainy season in the Yangtze River Basin has been demonstrated by decreased Fe (II) concentrations (Schaefer et al., 2016). Although the notable increase in $\mathsf { N O } _ { 2 } ^ { - }$ concentrations during monsoon suggests a shift to more oxidizing conditions, further analyses of redox active species, such as Fe (II) and S (II), would be needed to be certain of this redox shift.

# 3.4. Monsoonal influence on DOM absorbance

Our results also showed that changes to the UV absorbance of DOM were related to the monsoon. A statistically significant $\left( \mathtt { p } < 0 . 0 5 \right)$ twofold decrease in $\mathsf { S U V } \mathsf { A } _ { 2 5 4 }$ (from $4 \mathrm { L } / \mathrm { m g \cdot m }$ to $2 ~ \mathrm { L } / \mathrm { m g } { \cdot } \mathrm { m } ,$ ) was observed between pre-monsoon and during monsoon samples throughout the shallow aquifer (Fig. 9, Table S6). This would suggest that the aromaticity of DOM (Weishaar et al., 2003) may have diminished during monsoonal recharge. The DOM in high As groundwater in the Bengal Basin has been shown to be aromatic and humic-like (Mladenov et al., 2010, 2015; Kulkarni et al., 2017; Vega et al., 2017). Groundwater tapping the Pleistocene aquifer showed no change in $\mathrm { S U V } \mathsf { A } _ { 2 5 4 }$ , further reflecting the negligible effects of monsoonal recharge at greater depths. The decrease in $\mathsf { S U V } \mathsf { A } _ { 2 5 4 }$ in groundwater in the Holocene aquifer may be due to recharge by less aromatic rainwater and surface water inputs or reduction in the mobilization of aromatic DOM from sediments during reductive dissolution (Mladenov et al., 2010). Another influence on $\mathsf { S U V } \mathsf { A } _ { 2 5 4 }$ is the absorption by Fe (III) that may be present in the water sample. Our limited dataset lacking Fe speciation does not allow us to evaluate the influence of Fe on $\mathrm { S U V } { \tt A } _ { 2 5 4 }$ .

# 3.5. Discontinuous occurrence of As and DOM in Pleistocene aquifer

Occurrence of discontinuously distributed high dissolved As concentrations in typically low As groundwater (deeper groundwater tapping Pleistocene aquifer) in Bengal Basin is not uncommon (Burgess et al., 2010; Mukherjee et al., 2011; Erban et al., 2013), which has often been linked with draw-down of shallow As rich groundwater to greater depths. We found two deep tube-wells with dissolved $\mathsf { A s } _ { \mathrm { T } }$ concentrations exceeding national limits and one of which had extremely high $\mathsf { A } s _ { \mathrm { T } }$ concentrations $1 3 2 \mu \mathrm { g / L }$ and $7 1 5 \mu \mathrm { g / L }$ during pre-monsoon, 107 $\mu \mathrm { g } / \mathrm { L }$ and $6 8 6 \mu \mathrm { g / L }$ during-monsoon). The depths of these wells were $1 0 3 ~ \mathrm { m }$ and $1 0 9 ~ \mathrm { { m } }$ respectively tapping the Pleistocene aquifer (Kundu and Mandal, 2009) and were surrounded by low $\mathsf { A } s _ { \mathrm { T } }$ tube-wells at the same site Bele (within $0 . 5 \mathrm { k m }$ radius) and at site Radhanagar (at a distance of $1 \mathrm { k m }$ ). It should be noted that the highest dissolved $\mathsf { A } s _ { \mathrm { T } }$ concentration of $7 1 5 \mu \mathrm { g } / \mathrm { L }$ was ${ \sim } 5$ times higher than the average dissolved $\mathsf { A s } _ { \mathrm { T } }$ concentration of $1 3 8 \mu \mathrm { g / L }$ in the high As groundwater of this study (at Chakudanga and Shahispur sites tapping Holocene aquifer). Results of hydrochemical analysis showed that these two wells had lower $S 0 _ { 4 } ^ { 2 - }$ concentrations ${ \mathrm { \langle < 0 . 1 \ m g / L \rangle } }$ during both pre-monsoon and during monsoon, while groundwater wells at the other low As site, Radhanagar, had relatively higher $\mathrm { [ 6 \ m g / L }$ on average) $S 0 _ { 4 } ^ { 2 - }$ concentrations. Similarly, the concentrations of DOC $2 \mathrm { m g / L }$ each during pre-monsoon as well as during monsoon) and TDN (3.7 and $2 ~ \mathrm { m g / L }$ respectively during both seasons) were also higher as compared to the surrounding groundwater.

The reasons for the extremely elevated As in these two wells may be related to sedimentary geochemistry at these sites. Several studies have shown the presence of disconnected clay lenses within the aquifer (Mukherjee et al., 2007a, 2007b; Schaefer et al., 2016) that are high in organic matter and can supply the DOC necessary to fuel microbial reductive dissolution. While very little is known about the reactivity of such discontinuously existing DOC, our results show that the optical properties of DOM at these two wells was also contrastingly different than the surrounding groundwater. The surrounding groundwater had very low HIX (≤1) and humic:protein ratio (≤1), while these two wells exhibited HIX of 9 and humic:protein of 12, suggesting that the DOM at these locations was much more humic-like in nature. The HIX value and humic:protein ratio at these two wells in the Pleistocene aquifer, were 1.5 times higher than the average HIX (6) and twice as the humic:protein (6) found in typically high As groundwater tapping the Holocene aquifer.

# 4. Conclusions

This study examined the Holocene (shallow, high As) and Pleistocene (deep, low As) groundwaters in close proximity during dry season (May–June 2015) and after the monsoonal recharge began (July–August 2015). Spatial sampling and analyses revealed that shallow groundwater with high dissolved $\mathsf { A } s _ { \mathrm { T } }$ was associated with high DOC, TDN, humic:protein and low $S 0 _ { 4 } ^ { 2 - }$ and C:N molar ratio. These results suggest that nitrogen rich humic-like DOM may play a role in mobilizing As.

Our results from $\mathrm { C l / B r }$ ratios and stable isotope analyses further demonstrate that monsoonal recharge has a mixed influence on groundwater quality and that the influence of monsoonal recharge was limited to the shallow aquifer. While our study found insignificant changes in dissolved $\mathsf { A } s _ { \mathrm { T } }$ during rapid monsoonal recharge, the increase in $\mathsf { N O } _ { 3 } ^ { - }$ and $\mathsf { N O } _ { 2 } ^ { - }$ concentrations in the shallow reducing aquifer during monsoonal recharge reflected a possible shift in redox conditions. We further observed that the decrease in $\mathrm { S U V } \mathsf { A } _ { 2 5 4 }$ in the shallow groundwater during monsoonal recharge may have been due to decreased dissolved Fe (III), which is light-absorbing and may precipitate under more oxidizing conditions, dilution with less aromatic recharge water, or reduction in the release of sedimentary aromatic DOM during monsoonal recharge. Future work on Fe speciation analyses would provide further insight into the influence of redox state changes on $\mathsf { S U V } \mathsf { A } _ { 2 5 4 }$ due to monsoonal recharge.

Our study also identified two deep wells with significantly high dissolved $\mathsf { A } s _ { \mathrm { T } }$ in typically low As areas tapping the Pleistocene aquifer. The presence of highly humic-like DOM at such isolated As hotspots has not been reported earlier and highlights the importance of DOM chemical character in better understanding As mobilization in deep Pleistocene aquifers, which are generally considered to be free of As contamination. Overall, the results of this study help in advancing our knowledge about DOM quality in groundwater of contiguous Holocene and Pleistocene aquifers and chemical transformations occurring mainly in the shallow aquifer of Bengal Basin, where monsoonal precipitation is a primary source of groundwater recharge.
