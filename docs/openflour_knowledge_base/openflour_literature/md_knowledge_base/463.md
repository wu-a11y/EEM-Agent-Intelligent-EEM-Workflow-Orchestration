# Abstract

Impacts of ozonation alone as well as an advanced oxidation process of ozone plus hydrogen peroxide $( \mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 } )$ ） on organic matter prior to and following biofiltration were studied at pilot-scale. Three biofilters were operated in parallel to assess the effects of varying pre-treatmenttypesanddosages. Conventionallytreatedwater (coagulation/flocculation/sedimentation) was fed to one control biofilter, while the remaining two received water with varying applied doses of $\mathrm { O } _ { 3 }$ or $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ . Changes in organic matter were characterized using parallel factors analysis (PARAFAC) and fluorescence peak shifts. Intensities of all PARAFAC components were reduced by pre-oxidation, however, individual humic-like components were observed to be impacted to varying degrees upon exposure to ${ { \mathrm O } _ { 3 } }$ or $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ .While the control biofilter uniformly reduced fluorescence of all PARAFAC components, three of the humic-like components were produced by biofiltration only when preoxidation was applied. A fluorescence red shift, which occurred with the application of $\mathrm { O } _ { 3 }$ or $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ ， was atributed to a relative increase in carbonyl-containing components based on previously reported results. A subsequent blue shift in fluorescence caused by biofiltration which received pre-oxidized water indicated that biological treatment readily utilized organics produced by pre-oxidation. The results provide an understanding as to the impacts of organic matter character and pre-oxidation on biofiltration efficiency for organic matter removal. Keywords: natural organic mater; fluorescence spectroscopy; biofiltration; ozonation; advanced oxidation processes; drinking water treatment 7

# 1 Introduction

Natural organic matter (NOM) present in source waters plays a major role in the efficacy of drinking water treatment processes. Organic matter reacts with commonly used disinfectants, such as chlorine, to form a range of disinfection by-products which present some degree of public health risk (Hrudey 2009). Furthermore, treatment processes including granular media or membrane filtration, coagulation, and disinfection, can be impeded by NOM (Baghoth et al. 2011).

Rapid biologically active media filtration (biofiltration） is becoming a more widely employed NOM reduction approach (Chen et al., 2016). Intentionally backwashing traditional rapid media filters without a disinfectant residual promotes a biological community on the filter media which has been found to reduce NOM as well as disinfection by-product precursors (Emelko et al. 2006). An atractive feature of biofiltration is the simplicity of operation and ease of adapting existing infrastructure. To improve biofilter performance， several enhancement strategies have been investigated, including pre-oxidation using ozone (Yan et al. 2010) as well as addition of nitrogen and phosphorous (Fu et al., 2017; Lauderdale et al. 2012).

Organic matter in natural source waters is comprised of humic and fulvic acids, proteins, and carbohydrates (Her et al. 2003). Due to the breadth of chemical characteristics encompassed within the definition of NOM, analysis techniques which provide information regarding specific sub-fractions as opposed to a measurement of total quantity are advantageous. It is anticipated that these distinct NOM sub-fractions would have varying responses to treatment and unique impacts on finished water quality.

Fluorescence spectroscopy is a popular method for the characterization of NOM as it has proven to be sensitive and provide good selectivity (Li et al. 2014). For this purpose, fluorescence data is typically collected as high dimensional excitation-emission matrices (EEM). Molecular structure is associated with specific excitation/emission regions, which have a high degree of commonality among individual NOM components, therefore making fluorescence spectra of natural waters highly convoluted (Persson and Wedborg 2O01). Dimensionality reduction and deconvolution using multi-way paralel factor (PARAFAC) analysis and similar approaches have been successfully applied to account for the overlapping nature of organic classes within high-dimensional EEMs (Murphy et al. 2011). While application of PARAFAC elucidates underlying components which vary in the data set, representation of specific organic compounds is not guaranteed. For example, similar components observed among several studies have shown inconsistencies (Ishii and Boyer 2012). In addition, the presence of all identified PARAFAC components in fractions obtained following resin-based separation (He and Hur 2015） emphasizes that NOM characterization and sub-fractions are unique to the analysis methodology employed (Chen et al., 2016). While each PARAFAC component likely represents several organic compounds, it provides a grouping of fluorophores that explain variance between samples within defined spectral regions, which are related to distinct molecular structures (Stedmon and Markager 2003).

This study uses fluorescence to characterize the impacts of ozonation and an advanced oxidation process (AOP) consisting of ozone plus hydrogen peroxide $\left( \mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 } \right)$ on organic matter prior to biofiltration, as well as the resulting impact on biofiltration performance. Given the inconsistencies in component characteristics across studies, reporting how they are impacted by engineered systems is intended to help clarify interpretation of NOM fluorescence. # 2.2 Fluorescence

Fluorescence spectra were collected using an Agilent Cary Eclipse fluorescence spectrophotometer (Mississauga, Canada). Optimal instrument setings were determined based on previous studies and in-house testing (Peiris et al. 2009). Excitation and emission wavelength ranges were $2 5 0 \mathrm { ~ - ~ } 3 8 0 \mathrm { ~ \ n m }$ $5 \ \mathrm { n m }$ increments)， and $2 5 0 \mathrm { ~ - ~ } 6 0 0 \mathrm { ~ \ n m }$ $2 \ \mathrm { n m }$ increments), respectively. Each sampling day, a fluorescence spectrum of Milli- $Q ^ { \mathbb { Q } }$ water was collected and subtracted from each sample to account for the solvent background. The spectrum for Mili- $\mathbf { \mathcal { Q } } \mathbf { \circled { R } }$ water was also used to apply Raman corrections at an excitation wavelength of $3 5 0 \ \mathrm { n m }$ and bandwidth of $5 \ \mathrm { \ n m }$ to report fluorescence intensities in Raman Units (RU) (Lawaetz and Stedmon 2009). Absorbance spectra between 250 and $6 0 0 \ \mathrm { n m }$ (1 nm increments） for each sample were recorded using an Agilent 8453 UV-Vis spectrophotometer (Mississauga, Canada) to be used to correct for possible inner filter effects.

Fluorescence EEMs were analyzed using parallel factors analysis (PARAFAC).A methodology as described by Murphy et al. (2013) was followed using the drEEM toolbox for MATLAB. Rayleigh and Raman scatter regions were removed for conformity to the linear assumptions required for PARAFAC. Several samples were identified as outliers through observation of sample leverages on the model and were removed. The validity of the PARAFAC model was established through several means. Spectral loadings of the components were observed to conform to general guidelines on how organic fluorophores signals appear (e.g. only one emission peak, no abrupt changes in loadings). Split-half validation was also carried out based on a randomized split of the dataset to form 3 unique comparisons of dataset halves. For each unique half an independent PARAFAC model was developed and the components were matched to all other combinations as well as the complete model. Finally, calculated model residuals were observed to be random with few minor peaks. Model results were reported as $\mathrm { F } _ { \mathrm { m a x } }$ values in RU.

# 2.3 Other organic measures

Dissolved organic carbon (DOC) was measured using a persulfate wet oxidation method as described in Standard Method $5 3 1 0 \mathrm { ~ D ~ }$ (APHA 2012) with an O-I Corporation Model 1010 TOC Analyzer (Collge Station, Texas, USA). Ultraviolet absorbance at $2 5 4 ~ \mathrm { n m }$ $\left( \mathrm { U V } _ { 2 5 4 } \right)$ was determined using a CE 3055 model spectrophotometer (Cecil Instruments, Cambridge, England) with a quartz cuvete following Standard Method 5910 B (APHA 2012). Biological activity on the filter media was characterized by measuring adenosine triphosphate (ATP） from a media sample collcted from the top 5 cm of the filter.ATP concentrations were assayed with a LuminUltra Deposit Surface Analysis kit (DSA-100C, Fredericton, NB) following the manufacturer's instructions.

# 3Results and Discussion

# 3.1 PARAFAC analysis of EEMs

Application of parallel factors analysis (PARAFAC) to a set of 110 samples yielded a 5 component model. The total number of components was finalized by an assessment of the loading values and split half validation (Murphy et al. 2013). Resulting loading plots for the 5 PARAFAC components are shown as Figure 2. Fluorescing components were then crossreferenced with the OpenFluor database (Murphy et al. 2014b) through comparison of spectra from other studies to provide further context to the results. All components from this dataset were found to match (similarity score as defined by OpenFluor $> 0 . 9 5$ ） with at least one other study thus providing a basis to subsequently examine the components in this study as well as future studies. References for the strongest matches are shown in Table 1 along with suggested identities. Analyses using PARAFAC do not necessarily lead to the identification of unique organic compounds (Ishii and Boyer 2012; He and Hur 2015),as such components should be taken as groupings relevant to the observed variance in the sample set. Average raw water Raman corrected values for C1 to C5 fluorescence were: 1.82, 3.33，1.85， 1.68,and 0.93 respectively.

Components 1 to 4 were all found to be forms of humic-like matter with unique characteristics leading to separation by PARAFAC. C1 and C2 were identified as humic-like fluorophores with common peak maxima at $\mathrm { e x } / \mathrm { e m } \ 2 5 0 / 4 3 4 \ \mathrm { n m }$ .C1 has a spectrum similar to syringaldehyde which is produced from the breakdown of lignin (Murphy et al. 2014a). A PARAFAC component similar to C2 as discussed by Stedmon and Markager (2005) indicates that C2 may be produced from microbial activity. However, C2 loadings also adhere well with Component 1 discussed by Ishii and Boyer (2012） who ascribe the fluorophore to terrestrial origins,as did OpenFluor references, with reported links to rainfall and runoff events. The longer emission wavelengths present in C3 suggest this component represents high molecular weight plant-derived material with more conjugated fluorescing molecules (Coble et al. 1998). Similar to C2, C4 loadings peak at shorter excitation and emission wavelengths indicate fewer aromatic structures or conjugated bonds, lower hydrophobicity, and an overall lower molecular weight (Ishii and Boyer 2012; Fellman et al. 2010). Previous studies using OpenFluor suggest C4 as being from a microbial source. Component 3 discussed in Ishii and Boyer (2012) has similar loading characteristics as C4, which possbly resulted from oxidized quinone-like structures. C5 was the only protein-like component identified which was observed to have excitation/emission maxima similar to tryptophan (ex/em 280/342 nm).

# 3.2 Impact of pre-treatment on Organic Matter Character

The present study was conducted over two trials, one where ozone was examined at two doses (2.9 or $3 . 4 \mathrm { m g / L }$ ）and a second where an AOP $\left( \mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 } \right)$ was used at two dosage ratios (0.5 or $1 . 0 \mathrm { ~ m g ~ H _ { 2 } O _ { 2 } / m g ~ O _ { 3 } ) }$ . All pre-treatments (conventional, $\mathrm { O } _ { 3 }$ ，and $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { ~ O } _ { 3 } )$ were observed to reduce the fluorescence of the five identified PARAFAC components; however, had varying impacts on individual components resulting in altered relative component distributions (Figure 3). Raw water characteristics were consistent during the study period showing the high relative proportion of C2 fluorescence (representing $4 1 \mathrm { ~ - ~ } 4 4 \%$ of the total $\mathrm { F } _ { \mathrm { m a x } }$ ). Other studies have similarly noted a relative high proportionality of C2 fluorescence in surface waters (Ishii and Boyer 2012; Murphy et al. 2014b). The least abundant component was tryptophan/proteinlike (C5) comprising $< 1 0 \%$ of total $\mathrm { F } _ { \mathrm { m a x } }$ . Conventional treatment using alum showed a decrease in C2 proportionality $( 3 5 \% )$ with a corresponding increase in C1 and C4.

Application of ${ { \mathrm O } _ { 3 } }$ and $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ resulted in a pronounced reduction of C2 to $< 2 5 \%$ ， while the relative proportion of C1 fluorescence increased. These two components share a similar excitation/emission maxima at approximately $2 5 0 / 4 3 5 \ \mathrm { n m }$ , however C1 includes a second excitation peak at $3 4 0 ~ \mathrm { n m }$ .Humic-like components with multiple excitation peaks are routinely identified using PARAFAC analysis (Lu et al. 2009). Based on chromatographic separation of fluorophores with multiple excitation peaks, Li et al. (2013) suggest that multi-excitation peaked PARAFAC components are likely independent of other fluorophores sharing one or more excitation/emission maxima. Occurrence of multiple excitation peaks with a common emission can be explained by fluorescence theory, where absorption resulting in excitation to any vibrational state is rapidly followed by internal conversion to the lowest singlet state prior to emission (Lakowicz 2013). C2 is unique as a second excitation band was not identified in the measured range, excluding the protein-like C5.

Highest reductions of C2 and C4 fluorescence suggest that humic-like components at shorter excitation/emission wavelengths could be more amenable to oxidation via $\mathrm { O } _ { 3 }$ and/or $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ . Since commonly shorter excitation/emission wavelengths are associated with lower molecular weight and less conjugated and aromatic fluorophores (Fellman et al. 2010), the observed results are somewhat contrary to other studies which report a preferential impact of $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ and ${ { \mathrm O } _ { 3 } }$ on high molecular weight organics (Sarathy and Mohseni 20O7; Swietlik et al. 2004). The possbility of only the fluorescence signal of high molecular weight and highly aromatic fluorophores being less impacted by oxidation should also be considered given suggestions that a critical fluorophore or relatively small number of moieties within a compound determines the overall fluorescence of the whole humic (Li et al. 2013). Furthermore, it should be emphasized that molecular weight of the fractions are not being measured directly and are inferred from their fluorescence fingerprint.

Comparison between the two pre-oxidation strategies shows significantly improved removal of C2 fluorescence by $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ when compared to ${ { \mathrm O } _ { 3 } }$ alone $( \mathbf { p } = 0 . 0 0 5 )$ and lower efficiency at reduction of C1 fluorescence $( \mathtt { p } = 0 . 0 1 4 )$ ). These two methods of oxidation are known to have different and complex reaction mechanisms with NOM (van Geluwe et al. 2011). Westerhof et al. (1999) reported specific ultraviolet absorbance (SUVA) to be a good indicator of NOM reactivity with OH radicals (OH) produced by AOPs and $\mathrm { O } _ { 3 }$ .The absorbance for calculation of SUVA is measured at $2 5 4 ~ \mathrm { n m }$ and therefore should correlate with all humic-like components which have excitation peaks at approximately $2 5 0 \mathrm { n m }$ . In addition, previous studies have shown that $\mathrm { U V } _ { 2 5 4 }$ is also a strong predictor of exposure of NOM to ·OH (Audenaert et al. 2013). The distinct reactivity of identified fluorophores points to improved selectivity of the EEM approach when compared to absorbance or SUVA which would not be able to distinguish these humic-like components. No significant differences ( $9 5 \%$ confidence） were observed between the reductions in SUVA by $\mathrm { O } _ { 3 }$ $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ ：

Significantly lower reductions of C1, C3, and C4 were noted at the $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 }$ to $\mathrm { O } _ { 3 }$ dose ratio of $1 . 0 \mathrm { m g } \mathrm { H } _ { 2 } \mathrm { O } _ { 2 } / \mathrm { m g } \mathrm { O } _ { 3 }$ $\mathbf { \rho } _ { \mathrm { { p } } } = 0 . 0 0 1$ , 0.001, 0.030). Stoichiometry dictates an ideal weight ratio of $0 . 3 5 \mathrm { H } _ { 2 } \mathrm { O } _ { 2 } / \mathrm { O } _ { 3 }$ for ·OH formation from $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 }$ initiated ${ { \mathrm O } _ { 3 } }$ decomposition (Acero and von Gunten

2001). Therefore, $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 }$ was likely in excess and a residual may have been present. In natural waters presence of organic mater can result in the promotion and inhibition of ·OH formation, altering the ideal $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } / \mathrm { O } _ { 3 }$ ratio (von Gunten 2003). Acero and von Gunten (2001） found increased relative exposure of components in the water matrix to ·OH when compared to $\mathrm { O } _ { 3 }$ with increasing dose ratios of $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } / \mathrm { O } _ { 3 }$ . The relative inefficiency of higher $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } / \mathrm { O } _ { 3 }$ dose ratios for removing specific fluorescence components may suggest scavenging of ·OH by $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 }$ prior to reactions with select NOM components. However, the AOP reaction mechanisms are complex and more focused studies on the role of $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } / \mathrm { O } _ { 3 }$ dose ratios are required. The significantly lower reduction of protein-like fluorescence (C5) $( \mathfrak { p } = 0 . 0 0 2 )$ at higher ozone doses suggests that overdosing ozone or excess $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 }$ may have some inhibitory impact on removal of specific fluorophores, possibly through quenching ·OH or $\mathrm { O } _ { 3 }$

As previously reported by Baghoth et al. (2011)， all fluorescing components were observed to be greatly reduced by ozonation and $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ $( > 5 0 \%$ removal) (Figure 3). This would be expected as ozonation is known to be most reactive with aromatic moieties (Swietlik et al. 2004), which are the same molecular structures associated with fluorescence emissions from NOM (Hudson et al. 2007). The compounds which are oxidized likely produce non-fluorescent by-products, which can no longer be detected easily in the EEMs, and may represent a potential challenge in utilizing this method.

# 3.3Pre-oxidation and biofiltration performance

Three biofilters receiving differing pre-treated water (conventional/settled water, $\mathrm { O } _ { 3 }$ ,and $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 } )$ were compared during this study. Impacts to each of the five components by either pre-oxidation or biofiltration were normalized to setled water values to allw for direct comparisons (Figure 4).

Removal of the 5 components was greater by oxidation than biofiltration. When only considering the biofilters, reduction in fluorescence intensity was always greater for the control filters which received only settled water ( $( 8 . 9 \mathrm { ~ - ~ } 2 3 . 4 \%$ reduction). This appears contrary to previous studies which have shown improved DOC removal when considering biofilters which were preceded with ozone or an AOP (Speitel et al. 2000; Volk et al. 1997; Yan et al. 2010). It is proposed that in part the biodegradable fraction resulting from oxidation is not necessarily captured by fluorescence and therefore may result in this disagreement. Corresponding DOC results show significant improvements $( \mathtt { p } < 0 . 0 5 )$ regarding biofiltration performance when applying a $2 . 9 ~ \mathrm { m g / L }$ dose of ${ { \mathrm O } _ { 3 } }$ or $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ pre-treatment at either dose ratio compared to conventional (settled water) (Figure 4).

Consistent increases in C2 and C4 fluorescence were associated with biofiltration, supporting the assertion that these components may originate from microbial sources. In addition， most pre-oxidation conditions results in an increase in C1 fluorescence postbiofiltration. These three components have emissions $< 4 5 0 ~ \mathrm { n m }$ which has been suggested to be correlated with higher H/C and lower $\mathrm { O } / \mathrm { C }$ ratios (Lavonen et al. 2015). Therefore the removal of C3,with emissions $> 4 5 0 \ \mathrm { n m } .$ ， implies biofiltration in this case targeted more oxygen rich fluorophores.

Increased production of C2 and C4 by biofiltration was associated with pre-ozonation (C2: 17.1 - 18.0%; C4: $1 5 . 6 - 1 6 . 1 \%$ average increase） compared to $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ (C2: 10.1 - $1 0 . 5 \%$ ;C4: $5 . 6 - 9 . 1 \%$ average increase). A combination of increased removal by $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ (Figure 3) with reduced addition from biofiltration (Figure 4) was observed. Overall reduction of C2 fluorescence was greater for $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ -BAC $( 9 1 . 4 \mathrm { ~ - ~ } 9 2 . 2 \%$ average reduction） when compared to ${ { \mathrm O } _ { 3 } }$ -BAC $( 7 4 . 3 - 7 6 . 0 \%$ average reduction) (Figure 5). The decreased production of

C4 fluorescence from biofiltration using $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ pre-treatment resulted in overall greater removals ( $7 0 . 1 \ - \ 7 0 . 6 \%$ AOP-BAC vs. 61.5 - $6 2 . 0 \%$ ${ { \mathrm O } _ { 3 } }$ -BAC) despite similar performance between the oxidants alone (Figure 3). The $\mathrm { F } _ { \mathrm { m a x } }$ of terrestrial humic-like component (C1) was also observed to increase following biofiltration with pre-oxidized water. Reduction in proteinlike fluorescence (C5) by biofiltration was only observed with pre-oxidation at high ozone doses or $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } / \mathrm { O } _ { 3 }$ ratios， suggesting a dependency on the concentrations of oxidants. Overall $\%$ removals (Figure 5） however, are similar between low and high ozone or $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } / \mathrm { O } _ { 3 }$ ratios. Previously it has been noted that high ozone doses or $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } / \mathrm { O } _ { 3 }$ ratios in pre-treatment were found to be less effective at reduction of protein-like fluorescence (Figure 3).

Concentrations of components schowhich increased in concentration due to biofiltration after $\mathrm { O } _ { 3 }$ or AOP (C1, C2, and C4) were conversely removed by biofiltration receiving settled Water without pre-oxidation. As such, it is thought that oxidization of the organic matter allows for the production of these components. Filters receiving pre-oxidized water were observed to have significantly higher (t-test, $9 5 \%$ confidence) adenosine triphosphate (ATP) levels when compared to the control biofilter $( 2 6 - 5 4 \%$ greater; approximate range of $1 0 0 - 5 0 0 \ \mathrm { n g \ A T P / g }$ media). Increased biomass, as inferred by ATP levels, resulting from pre-oxidization may result in the production of select humic-like fluorophores. The biological availability of C2 ( $1 8 - 2 1 \%$ removal without pre-oxidation) is contrary to findings from several studies reviewed by Ishii and Boyer (2012). Distinct trends could be occurring within the same component as it is possible that PARAFAC analysis potentially lumps several types of organics into one component (He and Hur 2015).

In addition to PARAFAC analysis, a characterization approach of observing peak shifts has been used to identify changes in the character of NOM following oxidation (Volk et al. 1997;

Zhang et al. 2008). Emission spectra at excitation wavelengths of $2 5 0 ~ \mathrm { n m }$ ， $3 1 0 ~ \mathrm { { n m } }$ ,and $3 4 0 ~ \mathrm { n m }$ were selected to observe conformational changes. Excitation wavelengths were selected based on the location of peaks in the identified PARAFAC components.

Averaged peak maximums are reported in Table 2. Paired t-tests were performed ( $9 5 \%$ confidence level) to determine if a shift was significant due to treatment (e.g. ozonation or biofiltration). Significant red shifts due to $\mathrm { O } _ { 3 }$ and $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ were observed for all excitation scans. Red shifts in humic-like fluorescence have been ascribed to relative increase in carbonylcontaining structures in the fluorophores, including hydroxyl, alkoxyl, and amino groups (Swietlik and Sikorska 2O04). A red shift might also have occurred due to a relative increase in molecular weight or aromaticity (Chen et al. 2003); however, due to application of a strong oxidant this was thought to be unlikely. A relative increase in carbonyl-containing structures for specific fluorophores due to ozonation has been previously reported (Zhang et al. 2008; van Geluwe et al. 2011). Interestingly, biofiltration was observed to cause a blue shift back to settled water characteristics. However, the back and forth shift was not observed in the control biofilter, suggesting that carbonyl rich humics produced by O3 or $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ are more readily utilized by the microbial community. Increased biomass (as measured using ATP) following pre-oxidation may result from the availability of more labile organic mater. This conforms to suggestions that microbial processes will favourably utilize nonaromatic components of NOM (Hur et al. 2009).

As a result of the variable impacts of pre-oxidation and biofiltration on fluorescence components, the efluent water had a unique distribution of organic fluorophores (Figure 5). No significant differences $( \mathtt { p } > 0 . 0 5 )$ ）on the basis of SUVA between $\mathrm { O } _ { 3 }$ -BAC and $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ -BAC treatment were observed. Significantly improved removal of DOC and $\mathrm { U V } _ { 2 5 4 }$ was evident for the $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ -BAC compared to ${ { \mathrm O } _ { 3 } }$ -BAC at either dose ratio $( \mathbf { p } \le 0 . 0 0 1 )$ , reflecting the general trends observed in C1, C2, C4,and C5 fluorescence. However, use of these bulk measures neglects the consistent change to C3 fluorescence for all oxidant type and doses as well as preferential impact on certain organic fractions (e.g. C2).

The change in NOM character and proportional distribution of fluorescence components was further explored by grouping humic-like fluorophores to emissions $>$ 450 nm (C3) and $< 4 5 0$ nm (C1, C2, C4) (Figure 6). Through relating fluorescence in these two regions to results of ultrahigh resolution mass spectrometry Lavonen et al. (2015) reported the oxidation state of carbon to be $\geq 0$ for fluorophores emiting $> 4 5 0 ~ \mathrm { n m }$ and $\leq 0$ for organic material emiting $< 4 5 0$ nm. Furthermore the double bond equivalency per carbon was also higher for fluorescence emissions $> 4 5 0 ~ \mathrm { n m }$ . Conventional treatment (setted water) and subsequent biofiltration does not alter the proportionality of fluorophores above/below emissions of $4 5 0 ~ \mathrm { n m }$ ， $\mathrm { O } _ { 3 }$ or oxidation via $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ results in increased proportionality of humic-like material with higher emissions and therefore based on previously reported correlations (Lavonen et al. 2O15） an inferred increase in oxidation state and double bond equivalency per carbon. This observation supports the hypothesis that the red shift observed due to oxidation was caused by a relative increase in carbonyl containing functional groups.

In this set of experiments， biofiltration of ozonated water restored fluorescence characteristics comparable to conventional treatment or the control biofilter (without preoxidation). However, humic-like material in the effluent of $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ -BAC was observed to have a relatively higher proportion of matter with an inferred carbon oxidation state $\geq 0$ .This unique character of $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } \ + \ \mathrm { O } _ { 3 }$ -BAC treated water may have implications for increased chlorination disinfection by-product formation potential (Lavonen et al., 2013).

# 4Conclusions

Characterization of the organic matter using PARAFAC analysis identified 4 independent humic-like components (C1 - C4) and one protein-like component (C5). While the analysis provides a degree of fluorophore grouping based on explained variance, it may not be directly infer that the derived components are representative of individual organic compounds. The fluorescence-PARAFAC approach was found to identify distinct reactivity of fluorescence components with ${ { \mathrm O } _ { 3 } }$ or $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ as well as differing removal by the biofilters.

Pre-oxidation was observed to decrease the reduction of the fluorescence components by biofiltration alone compared to biofiltration receiving only setted water. However, removal of the components was much greater for biofilters with pre-oxidation when considering the impact of the oxidants. Overall reductions of most humic-like and protein-like fluorescence were observed using a combination of $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ and biofiltration compared to $\mathrm { O } _ { 3 }$ -BAC and a conventional BAC receiving setted water. Analysis of shifts in fluorescence peaks provided some insight into changes caused by pre-oxidation. As a result of ${ { \mathrm O } _ { 3 } }$ or $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ , a red shift was seen at excitation wavelengths of 250, 310, and $3 4 0 ~ \mathrm { n m }$ ， which was attributed to a relative increase in the abundance of carbonyl-containing structures. Biofiltration in all cases resulted in profiles that were analogous to pre-ozonation/AOP maxima, suggesting that the presence of carbonyl-containing structuresare more bioavailable orassociated with increased biodegradability of NOM.

Fluorescence spectroscopy presents itself as a useful method for characterizing the impacts of oxidation by $\mathrm { O } _ { 3 }$ or $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 } + \mathrm { O } _ { 3 }$ on NOM composition with improved selectivity over DOC，SUVA，or absorbance at $2 5 4 ~ \mathrm { \ n m }$ ：Furthermore， this approach demonstrates the complexity of understanding how NOM character both influences and is impacted by biofiltration. Future steps relating fluorophores to operational parameters are needed to beter inform drinking water treatment process optimization.
