The biogeochemical processing of dissolved organic matter (DOM) in inland waters is inherently related to its molecular structure and ecological function. Controlled bioassays are a valuable tool to analyze these relationships, but are seldom conducted and compared at temporal scales that typically prevail in natural inland waters. Here we incubated water from six boreal lakes in the dark and examined changes to the initial fluorescence and absorbance after 3.5 years. We identified five fluorescence components with parallel factor (PARAFAC) analysis $\mathrm { ~ \mathcal { C } _ { C } , \ d C _ { M } , \ d C _ { A } , \ d C _ { X } }$ and $\mathsf { C } _ { \mathrm { T } }$ ) and found a consistent change in the relative intensity of two dominant PARAFAC components (increase in ${ \mathsf { C } } _ { \mathsf { A } } { : } { \mathsf { C } } _ { \mathsf { C } } ,$ corresponding to Peak A:Peak C), commonly found in lake water, that represent terrestrially-derived DOM. Surprisingly, we only found minor changes to specific absorbance (SUVA), and did not find any changes to other spectral indexes including the fluorescence index, humification index and freshness index. By incorporating lakes spanning a wide range of initial total organic carbon concentrations (3.7 to $3 2 . 5 \mathrm { m g L ^ { - 1 } } .$ ), water residence times, and spectral characteristics (e.g. SUVA 1.13 to $3 . 7 7 \mathrm { L } { \cdot } \mathrm { m g } \mathrm { C } ^ { - 1 } { \cdot } \mathrm { m } ^ { - 1 } ,$ ), we found that the relative intensities of two humic-like peaks were the most revealing of changes to DOM structure during dark incubations. We also verified that inner filter effects were adequately corrected within the concentration range of incubated samples. Thus, the processing of DOM under dark conditions, including microbial decomposition and flocculation, may have a greater influence on the humic-like peaks, particularly $\mathsf { C } _ { \mathsf { C } }$ (Peak C), with negligible changes to more commonly used spectral indexes.

# 1. Introduction

Dissolved organic matter (DOM) within lakes is a mixture of poorly defined degradation products largely of terrestrial origin that can influence biological productivity, nutrient availability, light penetration and buffering capacity (Driscoll et al., 1994; Perez-Fuentetaja et al., 1999; Scully and Lean, 1994; Williamson et al., 1999). Along with growing recognition of the importance of carbon fluxes involving freshwater ecosystems in the global carbon cycle (Aufdenkampe et al., 2011; Battin et al., 2009; Cole et al., 2007), several studies have made progress to establish links between the structural composition of DOM and its origin and function in soils and aquatic ecosystems (Cabaniss et al., 2000; Kaiser et al., 1997; Kalbitz et al., $2 0 0 3 \mathrm { a }$ ; McKnight et al., 2001; Tipping, 1998). The structural composition of DOM is however dynamic during passage through inland waters (Wu et al., 2007), since several interdependent processes including microbial processing, photo-degradation and flocculation, contribute to its cycling and loss (Amon and Benner, 1996; Bertilsson and Tranvik, 2000; Molot and Dillon, 1997; Morris et al., 1995; von Wachenfeldt and Tranvik, 2008). Additionally, phytoplankton and attached vegetation continuously add internally produced DOM to the water column (Baines and Pace, 1991; Strom et al., 1997).

Studies have shown that microbial degradation of DOM is enhanced by photolysis, whereby DOM is transformed to less colored material (Bertilsson and Tranvik, 2000; Moran et al., 2000; Stedmon and Markager, 2005b). Other studies have observed the sunlightmediated transformation of diagenetically fresher DOM into less biologically available compounds with higher absorption properties, by a process called photo-humification (Obernosterer et al., 1999; Ortega

Retuerta et al., 2007; Tranvik and Bertilsson, 2001). However, the cycling of DOM in lakes also includes processes occurring under dark conditions. The percent of incident light able to penetrate through the water column decreases quickly with depth due to absorption, scattering and reflection, and is highly variable between lakes. This light attenuation is negatively related to colored DOM concentration, and the photochemically active short wavelength radiation (UV-B, UV-A, near-UV visible radiation) is attenuated in the uppermost part of the water column (Morris et al., 1995; Perez-Fuentetaja et al., 1999; Scully and Lean, 1994). Accordingly, light-induced processing of DOM in lakes with greater concentrations of colored DOM is restricted to the uppermost layers of the water profile. In the case of humic lakes typical of the Swedish boreal landscape, photochemical effects on bacteria and DOM can only be detected in the upper meter of the water column during high summer conditions (Lindell et al., 1996). Considering the deeper layers of the lake profile, and periods of the year with less light availability including ice cover, it is relevant to understand how light-independent processes contribute to the transformation and degradation of lake DOM.

Studies have used controlled bio-assays to track changes in DOM characteristics over time (Lindell et al., 1995), with a direct comparison of photo-degradation and dark experiments run in parallel (Moran et al., 2000; Vähatalo and Wetzel, 2004). However, bioassays have been largely restricted to short time scales of several weeks to months. Thus, there is an inherent assumption that observed changes to DOM structure, or selective loss of certain characteristics may continue, even at longer time scales. DOM resistant to biodegradation processes has been found to contain a high content of stable aromatic structures, including lignin compounds which are biopolymers found exclusively within terrestrial vascular plants (Kalbitz et al., 2003b).

Spectroscopic techniques provide a relatively simple method of tracking changes to the bulk chemical composition of DOM. In particular, the optically active fraction of DOM that fluoresces is inherently related to the chemical and structural composition of DOM. While many studies examine absorbance spectra, there is a greater amount of spectral information in 3-dimensional fluorescence scans. The interpretation of excitation–emission-matrices (EEM) derived from fluorescence spectroscopy has developed substantially with the aid of a powerful multivariate statistical approach called parallel factor analysis (PARAFAC), the DOM-fluor toolbox (Andersson and Bro, 2000), and a helpful tutorial (Stedmon and Bro, 2008). PARAFAC deconstructs and identifies underlying fluorescing regions of the EEMs in a particular dataset and thus allows for further interpretation of these identified fluorescence components. While knowledge regarding the origin and molecular structure of fluorescing DOM signatures is still evolving (Cory and McKnight, 2005; Hernes et al., 2009; Maie et al., 2007; McKnight et al., 2001), distinct regions of the EEM have been associated with DOM containing protein-like structures that are largely of microbial or phytoplankton origin, and humic-like structures largely of terrestrial origin (Coble, 1996; Coble et al., 1990; Fellman et al., 2010; Parlanti et al., 2000; Yamashita and Tanoue, 2003). Consequently, the protein-like fluorescence has been related to DOM bioavailability (Guillemette and del Giorgio, 2011; Lønborg et al., 2010). While the use of fluorescence spectroscopy develops, there is some advantage to clearly identifying spectral signatures reflecting specific DOM biogeochemical processes by using controlled laboratory conditions. These signatures can potentially help provide a basis for unraveling spectral patterns observed in nature. While a previous manuscript attempts to link spectroscopic characteristics to predicting the rate of degradation using a continuum model (Koehler et al., 2012), here we focus on changes to the structural characteristics of DOM during a dark degradation.

The objective of this study was to identify if there are selective changes in the relative abundance of fluorescence components in lake water DOM induced by several years of dark incubation. We also examined changes to several other commonly used absorbance and fluorescence metrics. We expected the natural variability of initial lake water DOM composition to influence decomposition, and thus six lakes with a broad gradient of initial DOM concentrations (3.7 to $3 2 . 5 \mathrm { m g L } ^ { - 1 }$ ), water residence times, and spectral characteristics were included.

# 2. Methods

# 2.1. Study sites

The study was conducted for six boreal lakes positioned between $5 9 ^ { \circ } 4 4 ^ { \prime }$ to $6 4 ^ { \circ } 0 7 ^ { \prime }$ N and $1 5 ^ { \circ } 2 3 ^ { \prime }$ to $1 8 ^ { \circ } \ 4 7 ^ { \prime } \ \mathrm { E }$ , in south-central Sweden (Table 1). Across lakes, average depth ranged from 1.3 to $6 . 0 \mathrm { m }$ , pH ranged from 5.3 to 8.2, total phosphorus ranged from 8 to $4 7 \mu \mathrm { g } \mathrm { L } ^ { - 1 }$ , and total nitrogen ranged from 0.5 to $1 . 7 \mathrm { m g L } ^ { - 1 }$ . Initial total organic carbon concentrations ranged from 3.7 to $3 2 . 5 \mathrm { m g L } ^ { - 1 }$ and theoretical water residence time ranged from $\sim 3$ weeks to ${ > } 4$ years (Brunberg and Blomqvist, 1998). An additional nearby lake (Hanelundssjön, $5 9 ^ { \circ } 5 4 ^ { \prime } \mathrm { N } 1 7 ^ { \circ } 0 8 ^ { \prime } \mathrm { E } )$ considered to be representative of naturally humic waters, was sampled to test the efficacy of the inner filter corrections. Hanelundssjön is a small shallow lake $( 1 { - } 2 \mathrm { m } )$ , with a catchment area primarily of forest and wetlands (Brunberg and Blomqvist, 1998). The climate of this region in Sweden has mean annual temperatures of $2 { - } 6 ^ { \circ } C ,$ and mean annual precipitation of $6 0 0 { - } 9 0 0 ~ \mathrm { m m }$ (Swedish Meterological and Hydrological Institute S., 2011). Catchment areas of lakes included in the incubation study are dominantly coniferous forest except for one lake (Valloxen), which has a higher proportion of agriculture.

Water samples were filtered through a coarse $4 5 \mu \mathrm { m }$ mesh to avoid removal of bacteria and bacterivores, hence allowing for a microbial loop with re-mineralization of nutrients throughout the incubation. Lake water samples were brought to initial oxygen concentrations of $9 . 8 \pm 0 . 5 \% 0 _ { 2 }$ by bubbling with synthetic air $( 2 0 . 2 \pm 0 . 5 \% 0 _ { 2 } )$ ) and were transferred headspace-free into pre-combusted $4 0 \mathrm { m L }$ glass bottles with Teflon coated septa. Samples were incubated in dark water baths at $2 0 ~ ^ { \circ } \mathrm { C }$ for approximately 3.5 years (between 1276 and 1331 days). Oxygen concentration was measured in sealed bottles through the septa at the end of the incubation to ensure that incubated water did not become anoxic using 20 randomly chosen bottles. Further details about the bioassay study are elsewhere (Koehler et al., 2012). All samples that were harvested on day 0 were run in triplicate, while all control samples harvested on the final day of the incubation were in duplicate or triplicate.

# 2.2. DOM analysis

All samples were analyzed for total organic carbon (TOC) concentration using a Sievers 900 total organic carbon analyzer without further filtration (General Electric Analytical Instruments, Manchester, UK). Boreal lakes included in this study typically have low concentrations of particulate material, thus TOC here is comparable to DOC, within a few percent. While the quantification of organic matter concentration is based on the carbon content, spectroscopic analysis includes the entire DOM pool, thus we used the term DOM throughout the manuscript for consistency. Optical analysis was done without any further filtration beyond the initial $4 5 \mu \mathrm { m }$ mesh.

All EEMs were blank-subtracted using the EEM of pure water run on the same day. Manufacturer supplied correction factors were used to correct excitation and emission intensities for instrumentspecific biases. These correction factors were applied for both lamp excitation and emission biases, since spectrophotometer components including gratings, detectors, mirrors, lenses and beam splitters have instrument specific responses. Spectra were corrected for the absorption of incident and emitted light due to obstructions from colored organic matter suspended within the sample cuvette, also known as the inner filter effect using separate absorbance scans (addressed more in Section 2.6) (Lakowicz, 2006; McKnight et al., 2001). The fluorescence intensity was adjusted to Raman units by dividing by the Raman area of pure water integrated at an excitation of $3 5 0 \mathrm { n m }$ , and over an emission range of 380 to $4 2 0 \mathrm { n m }$ (Lawaetz and Stedmon, 2009). All samples were run in signal (S) mode and corrected for lamp excitation reference intensity (R) mode following measurements by scaling to the recorded lamp intensity. Some residual traces of a Rayleigh peak were removed prior to PARAFAC analysis by transforming negative values to zero.

# 2.4. PARAFAC modeling

Individual fluorescing components in lake water EEMs were determined using the DOM-fluor toolbox (MATLAB $\textsuperscript { \textregistered }$ 7.7.0, The MathWorks, Natick, USA, 2008) and guidance from the appendix of Stedmon and Bro (2008). Model components were generated from the spectra of 67 samples. Prior to PARAFAC modeling, the second order scatter was removed by converting this region to not-anumber, starting at an emission wavelength of $5 0 0 \mathrm { n m }$ . In our dataset, the region with an emission above $5 6 0 \mathrm { n m }$ was free of fluorescence signal, and thus was cut off to reduce the size of the EEM and assist the modeling process. First order scatter was removed by inserting a band of zeros where excitation was greater than emission, starting at $2 9 8 \mathrm { n m }$ . After running a series of models ranging from 4 to 9 component models with non-negativity restrictions whereby loadings plots are forced to be positive, we found that a 4 or 5 component model would sufficiently describe our dataset while minimizing residuals. Split-half validation was used to randomly divide the dataset into sub-sets which could then be tested for validation using a series of 10 random initializations. Both splits of the 4 and 5 component models were validated, while artifacts were apparent in the excitation and emission loadings of the 6 component model. After examination of the residuals of the measured and modeled EEMs it was clear that a 5 component model adequately described all fluorescing regions of the 67 EEMs. While three of the five components had multiple excitation loadings, all emission loadings had smooth uni-modal peaks as desired for fluorophores (Andersen and Bro, 2003).

Due to the large time lapse between initial and final analysis (3.5 years), some drifting in lamp intensities and instrument conditions could contribute to uncertainty associated with directly comparing the fluorescence intensities of components between initial and final days. For this reason, we choose to focus our interpretation of results to the comparison of relative intensities of two major fluorescing peak $( \mathsf { C } _ { \mathsf { A } } { : } \mathsf { C } _ { \mathsf { C } } )$ , and the relative abundance of any particular PARAFAC component (e.g. $\% \mathrm { = } \mathrm { C _ { A } / ( C _ { C } + C _ { M } + C _ { A } + C _ { X } + C _ { T } ) } .$ (components identified in Table 2).

The fluorescence index (FI) was calculated as the ratio of λEm at 470 and $5 2 0 \mathrm { n m }$ at an λEx of $3 7 0 \mathrm { n m }$ (Cory et al., 2010). The freshness index (β:α) was calculated as the ratio of λEm at $3 8 0 \mathrm { n m }$ divided by the λEm maximum between 420 and 435, at an λEx of $3 1 0 \mathrm { n m }$ (Parlanti et al., 2000; Wilson and Xenopoulos, 2009). We calculated the humification index (HIX) (Zsolnay et al., 1999) using the modified method (Ohno, 2002) whereby, at an λEx of $2 5 4 \mathrm { n m }$ , the area under the spectra at λEm from 435 to $4 8 0 \mathrm { n m }$ is divided by the sum of the area at λEm of 435 to $4 8 0 \mathrm { n m }$ and 300 to $3 4 5 \mathrm { n m }$ .

# 2.5. UV–visible absorbance

Absorbance spectra were measured across a wavelength range from 200 to $6 0 0 \mathrm { n m }$ , at $1 \mathrm { n m }$ intervals using a Lambda 40 UV–visible spectrophotometer (Perkin Elmer, Waltham, USA). Samples were measured in a $1 \mathrm { c m }$ quartz cuvette and distilled water was used as the blank. The slope (S) of the absorbance spectrum was estimated for the region between 280 and $4 0 0 \mathrm { n m }$ . The shape of the spectra has been previously described by (Stedmon et al., 2000) as method 1, whereby $\mathsf { a } _ { \mathsf { N } }$ $( \mathfrak { m } ^ { - 1 } )$ is the absorbance coefficient at the wavelength $\lambda$ (nm), and ${ \sf a } _ { \Lambda 0 }$ is the absorbance at a reference wavelength, $\lambda _ { \sigma }$ $( 2 8 0 \mathrm { n m } )$ ). $S _ { 2 8 0 - 4 0 0 }$ was calculated as the slope of the exponential curve $( a _ { \lambda } = a _ { \lambda o } \ e ^ { S ( \lambda o - \lambda ) } )$ between 280 and $4 0 0 \mathrm { n m }$ , and provides a measure of how steep the decline in absorbance occurs with increasing wavelength. The DOM normalized absorbance at $2 5 4 \mathrm { n m }$ was calculated for the SUVA as previously described, and was expressed as $\mathrm { L } { \cdot } \mathrm { m g C } ^ { - 1 } { \cdot } \mathrm { m } ^ { - 1 }$ (Weishaar et al., 2003).

# 2.6. DOM concentration series

To demonstrate the effectiveness of the absorbance-based correction factor for inner filter effects within the concentration range of this study, we included one concentration series. DOM solutions spanning from an original concentration of $3 8 . 7 \mathrm { m g L } ^ { - 1 }$ , down to $4 . 0 \mathrm { m g L } ^ { - 1 }$ , were created by dilution from water collected at Lake Hanelundssjön using de-ionized water. Absorbance at $2 5 4 \mathrm { n m }$ $\left( \mathsf { A } _ { 2 5 4 } \right)$ for the concentration series spanned from 0.16 to 1.69. Spectra were compared with and without corrections after normalizing spectral intensities (Raman units) to $\mathsf { A } _ { 2 5 4 }$ .

# 3. Results

# 3.1. DOM

Initial TOC concentrations of the lakes ranged from 3.7 to $3 2 . 5 \mathrm { m g L } ^ { - 1 }$ and $\mathsf { A } _ { 2 5 4 }$ ranged from 0.11 to 1.23 (in a $1 \mathrm { c m }$ cell) (Table 1). By the final day of the study (3.5 years), between 55 and $6 8 \%$ of the initial DOM remained (Table 1). Oxygen concentration in final samples was $8 . 7 { \pm } 0 . 1 \ \mathrm { m g } \mathrm { L } ^ { - 1 }$

# 3.2. Fluorescence components

A total of five fluorescing components (C) were identified from PARAFAC analysis (Fig. 1). The location of maximum peak intensities for PARAFAC components were similar to Peaks commonly referred to in the literature (Peaks A, C, M and T) (described in Table 2). Thus, component names $\mathrm { \Delta C _ { C } \ C _ { M } \ C _ { A } \ C _ { X } }$ and $\mathsf { C } _ { \mathrm { T } }$ ) were chosen to reflect these common peak names, with the exception of $\mathsf { C } _ { \mathsf { X } } ,$ which is unidentified. There was a large amount of variability between the abundance of components between lakes. The most dominant component on average (mean $\pm$ standard deviation) across all sites was $\mathsf { C } _ { \mathsf { A } }$ $( 3 7 . 8 \pm 7 . 5 \% )$

followed by $\mathsf { C } _ { \mathsf { C } }$ $3 1 . 6 \pm 5 . 5 \%$ , $\mathsf { C } _ { \mathrm { M } }$ $( 1 9 . 4 \pm 2 . 7 \% )$ , $\varsigma _ { \mathrm { X } }$ $( 6 . 9 \pm 1 . 5 \% )$ , and $\mathsf { C } _ { \mathrm { T } }$ $( 4 . 4 \pm 3 . 0 \% )$ (Table 3). The single most visually obvious alteration between initial and final EEMs was the reduced intensity of $\mathsf { C } _ { \mathsf { C } }$ (Peak C) relative to $\mathsf { C } _ { \mathsf { A } }$ (Peak A), which was detectable across all lakes (Fig. 2). The average ratio of $\mathsf { C } _ { \mathsf { A } } { : } \mathsf { C } _ { \mathsf { C } }$ at the start of incubation spanned a broad range from 0.16 to 0.77 across the six sites, however by the final day of the incubation (3.5 years) the ratio had consistently increased, to range from 0.55 to 1.77 (Fig. 3a). The change in ${ \mathsf { C } } _ { \mathsf { A } } { : } { \mathsf { C } } _ { \mathsf { C } }$ ranged from 0.22 to 1.38 across the six lakes. After 3.5 years of dark incubation there were several shifts to the relative abundance of PARAFAC components (Table 3). Based on paired $t$ -tests for sample means, $\% \ C _ { \mathrm { A } }$ increased, while $\% \mathrm { C _ { C } , C _ { M } }$ and $\mathsf { C } _ { \mathsf { X } }$ decreased (Table 1) $\scriptstyle ( p < 0 . 0 5 )$ , however $\% \mathrm { C _ { T } }$ was an exception and remained unaffected over time $\scriptstyle { p = 0 . 1 6 }$ . We found closely comparable trends in PARAFAC components $\left( \mathsf { C } _ { \mathsf { A } } \mathrm { : C _ { C } } \right)$ with Peak A:Peak C (Fig. 3b), whereby the intensity of Peak A is larger than Peak C after the incubation.

# 3.3. Spectral indexes

Of the spectral metrics used in this study $( S _ { 2 8 0 - 4 0 0 }$ , SUVA, FI, $\beta / \alpha$ and HIX), a change was only observed in two metrics (FI and SUVA) after the 3.5 year incubation period based on paired t-tests (Table 1), despite the wide span of starting concentrations and water residence times. The final mean $\pm$ standard deviation) FI increased for all lakes from 1.32 $( \pm 0 . 0 5 )$ to 1.36 $( \pm 0 . 0 5 )$ $\scriptstyle ( p < 0 . 0 1 )$ . Since a difference in FI of at least 0.10 would be required to suggest a change in source (McKnight et al., 2001), we consider the observed change in FI to be negligible. We found a statistically significant increase in the SUVA from a mean of 2.83 $\left( \pm 1 . 0 5 \right)$ to 3.34 $( \pm 0 . 9 2 )$ $\mathsf { L } { \cdot } \mathsf { m g C } ^ { - 1 }$ · $\mathrm { m } ^ { - 1 }$ across all sites $\scriptstyle ( p < 0 . 0 1 )$ , which corresponds to an increase in aromaticity (Weishaar et al., 2003). Based on the linear regression previously established between SUVA and aromaticity (Weishaar et al., 2003), the initial aromaticity across the 6 sites shifted from $2 2 \%$ to $2 5 \%$ after 3.5 years. The relationship between FI and aromaticity is not as tight as with SUVA, and thus it has been cautioned that a direct conversion from FI to aromaticity is not possible (McKnight et al., 2001). Since the FI is an indicator of precursory material, all samples maintained a terrestrially-derived signature.

# 3.4. DOM concentration series

The DOM concentration series ranged from 4.0 to $3 8 . 7 \mathrm { m g L } ^ { - 1 }$ which extends beyond the maximum concentration of samples included in the incubation study $( 3 2 . 5 \ : \mathrm { m g } \ : \mathrm { L } ^ { - 1 } $ ), and likewise the $\mathsf { A } _ { 2 5 4 }$ nm spanned from 0.16 to $1 . 6 9 \mathrm { c m } ^ { - 1 }$ , which is also well beyond the maximum $\mathsf { A } _ { 2 5 4 }$ for incubated samples (1.23). When fluorescence spectra from the dilution series were compared at λEm of $4 4 8 \mathrm { n m }$ it was clear that the fluorescence intensities of more concentrated samples were reduced relative to the diluted samples, however after applying the inner-filter correction to the dilution series, intensities across the concentration gradient were comparable (Supp. Fig. 2). As can be expected, the greatest suppression of fluorescence intensity was observed at lower wavelengths, affecting $\mathsf { C } _ { \mathsf { A } }$ more so than $\complement _ { \mathrm { C } } .$ Without the inner filter correction, the ${ \mathsf { C } } _ { \mathsf { A } } { : } { \mathsf { C } } _ { \mathsf { C } }$ spanned from 0.52 for the most optically dense sample $( 3 8 . 7 \mathrm { m g } \mathrm { L } ^ { - 1 }$ ), to 1.50 for the most dilute sample $( 4 . 0 \mathrm { m g L } ^ { - 1 }$ ), however the mean ( $\pm$ standard deviation) $\mathsf { C } _ { \mathsf { A } } { : } \mathsf { C } _ { \mathsf { C } }$ for inner-filter corrected spectra was $1 . 8 1 \pm 0 . 1 1$ , and was greatest at higher concentrations (Fig. 5a). Thus, the increase in ${ \mathsf { C } } _ { \mathsf { A } } { : } { \mathsf { C } } _ { \mathsf { C } }$ we observed during the incubation (with decreasing DOM concentrations) could not be attributed to inner filter effects. The variability in SUVA observed across the dilution series ranged between 4.06 and $4 . 4 0 \mathrm { L } { \cdot } \mathrm { m g } \mathrm { C } ^ { - 1 } { \cdot } \mathrm { m } ^ { - 1 }$ with a mean ( $\pm$ standard deviation) of $4 . 2 8 \pm 0 . 1 3 \mathrm { L } { \cdot } \mathrm { m g } \mathrm { C } ^ { - 1 } { \cdot } \mathrm { m } ^ { - 1 }$ (Fig. 5b). We found the SUVA was slightly higher for more concentrated samples in the dilution series $( \mathsf { S U V } \mathsf { A } = 0 . 0 1$ $( \mathsf { D O M } ) + 4 . 1$ , $\mathrm { R } ^ { 2 } { = } 0 . 8 3$ ). However, since samples after the 3.5 year incubation had a lower DOM concentration but consistently increased SUVA, the change in SUVA observed over the incubation study could not be an artifact of concentration. The FI of the concentration series ranged between 1.19 and 1.23 (Fig. 5c), and thus the effect of DOM concentration between 3.7 and $3 2 . 5 \mathrm { m g L } ^ { - 1 }$ , was minor, spanning a maximum of $0 . 0 4 \mathrm { F I }$ units. While no significant change was detected in the HIX or freshness index over the incubation study (Table 1), both indexes had little variability across the DOM concentration series (not shown), with mean $\pm$ standard deviation) values of 0.96 $\left( \pm 0 . 0 1 \right)$ and 1.33 $\left( \pm 0 . 0 4 \right)$ , respectively.

# 4. Discussion

This study demonstrates that one of the most commonly identified DOM fluorescing regions, considered to be of terrestrial origin, commonly referred to as Peak C, and identified by PARAFAC analysis here as a component $\complement _ { \mathrm { { C } } }$ is preferentially lost relative to $\mathsf { C } _ { \mathsf { A } }$ (Peak A) over a 3.5 year period. While selective photo-degradation of DOM fluorescence has been previously documented (Moran et al., 2000), this study verifies that dark degradation can also selectively alter the structural composition of lake water DOM. Here, we have a unique opportunity to trace long-term relative changes to five fluorescence components identified from PARAFAC analysis, and several commonly used absorbance (SUVA, $S _ { 2 8 0 - 4 0 0 } )$ and fluorescence (FI, HIX, $\alpha / \beta \AA )$ ) metrics.

The most striking observation was the consistent preferential retention of $\mathsf { C } _ { \mathsf { A } }$ over $\mathsf { C } _ { \mathsf { C } }$ across all samples, irrespective of initial concentration or water age. Both $\mathsf { C } _ { \mathsf { A } }$ over $\complement _ { \mathrm { C } }$ encompass regions of the EEM that are often referred to as humic-like material of terrestrial origin. Similarly located peaks, Peaks A and C, are commonly reported peaks within the literature, (Coble et al., 1990, 1998; Fellman et al., 2010; Hudson et al., 2007). A common approach has been to compare changes to the relative abundance of humic-like peaks (Peaks A and C) with protein-like (Peaks T and B), however there is still little evidence of how Peak A properties might differ from Peak C. When combined, we found the humic-like peaks, $\mathsf { C } _ { \mathsf { C } }$ and $\complement _ { \mathsf { A } } ,$ to comprise the majority of the overall fluorescence, however the identification of structural characteristics unique to these two peaks remains elusive. Terrestrially-derived lignin have traditionally been considered to be recalcitrant components of DOM, along with tannins, cellulose and hemicellulose. Hernes et al. (2009) however identified that the fluorescing region associated with lignin overlaps with the region generally associated with aromatic amino acids, tryptophan (Peak T) and tyrosine (Peak B). In fact, the prominent fluorescing peaks (A and C) that are commonly associated with terrestrial sources were unable to predict the lignin content in the riverine study (Hernes et al., 2009). Thus, while $\mathsf { C } _ { \mathsf { A } }$ (Peak A) is not likely comprised of lignin, it remains unclear which class of organic compounds within $\mathsf { C } _ { \mathsf { A } }$ were more resistant to loss processes than $\complement _ { \mathrm { C } }$ in our study. Quinone-like fluorophores may account for up to $50 \%$ of the observed fluorescence in aquatic environments, whereby underlying fluorophore components identified by PARAFAC were found to be similar to reduced and oxidized quinones (Cory and McKnight, 2005). Studies have demonstrated that humic substances, and more specifically quinone moieties are involved in electron transfer during microbial respiration in anoxic conditions (Lovley et al., 1996; Scott et al., 1998). We found that oxidized quinones identified by Cory and McKnight (2005) have some resemblance to $\complement _ { \mathsf { A } } ,$ and likewise some reduced quinones and unidentified components had some resemblance to $\complement _ { \mathrm { C } }$ While the observed increase in ${ \mathsf { C } } _ { \mathsf { A } } { : } { \mathsf { C } } _ { \mathsf { C } }$ during a dark incubation study could suggest a transformation of reduced to oxidized quinones in anoxic condition, the bottles in this study remained well oxygenated throughout the 3.5 years. Thus, it is mechanistically unclear how quinone fluorescence would be modified during an incubation study in oxic conditions. Results from this study do suggest that DOM comprising $\mathsf { C } _ { \mathsf { C } }$ was more labile than $\mathsf { C } _ { \mathsf { A } }$ during long-term dark oxic conditions. Our findings of reduced Peak $\mathsf { C } _ { \mathsf { C } }$ relative to $\mathsf { C } _ { \mathsf { A } }$ are not unique. A marine study by Moran et al. (2000) examined the rate of carbon loss from an estuary by photo-degradation, biological degradation and both processes together. Their findings agree with ours, as they too found a slight preferential removal of Peak C over Peak A during a dark incubation, even within the relatively short time period of 51 days. The stark increase in $\mathsf { C } _ { \mathsf { A } } { : } \mathsf { C } _ { \mathsf { C } }$ we observed could be interpreted as the identification of two common fluorescing regions of lake water EEMs with differing light-independent loss rates. The selective loss of $\mathsf { C } _ { \mathsf { C } }$ suggests that this DOM material is of moderate degradability (at a 3 to 4 year time scale), whereas $\mathsf { C } _ { \mathsf { A } }$ may represent more recalcitrant DOM during light-independent DOM processing.

Interestingly, the change in fluorescence patterns observed under dark incubations was not consistent with patterns we found across the natural DOM-water residence time gradient (Fig. 4). While $\mathsf { C } _ { \mathsf { A } }$ : $\mathsf { C } _ { \mathsf { C } }$ increased consistently during the 3.5 years of dark incubation, $\mathsf { C } _ { \mathsf { A } } { : } \mathsf { C } _ { \mathsf { C } }$ decreased across lakes with increasing water residence times (Fig. 4b), and correspondingly low DOM concentration. The contrasting response between these two dominant fluorescence peaks may be the result of differing mechanisms in the controlled laboratory and under natural conditions in a lake. The negative relationship between DOM concentration and water residence time (Supp. Fig. 1) gives us reason to make an inherent assumption that sites with lower DOM and longer water residence times, such as Ljustjärn, contained DOM with a more progressed level of internal lake processing. In fact, this is supported by the lower humification index observed at lakes with longer water residence times (e.g. Ljustjärn, Table 1). This theory is consistent with other observations in Sweden of increasing in-lake loss of DOM with increasing water residence times (Algesten et al., 2004; Weyhenmeyer et al., 2012). Bacterial production and consumption have been linked to protein-like components and Peak T intensity (Guillemette and del Giorgio, 2011; Romera-Castillo et al., 2011), yet there is an ongoing need to learn more about the ecological applications of humic-like and protein-like peaks in freshwater systems (Fellman et al., 2010). The controlled 3.5 year laboratory incubation excluded photo-degradation and primary production since samples were in closed dark systems. Thus, results from our study largely reflected the continual microbial recycling of existing DOM, which likely decreased in terms of biological availability over time, along with flocculation processes. While flocculation is a potentially important loss process (von Wachenfeldt et al., 2008; von Wachenfeldt and Tranvik, 2008), we were unable to quantify relative losses to flocculation and respiration here. Thus, DOM losses are a combination of bacterial respiration and flocculation. Although conditions during long term incubations of contained samples inevitably deviate from the in situ conditions of lakes, the controlled conditions that can be achieved in confined samples allow observations that record processes that are otherwise not possible to demonstrate. Regardless of the opposing trends observed along an in situ continuum and the dark bioassay, the stark trends we observed in the bioassay study (Fig. 2) provide a foundation to monitor the ${ \mathsf { C } } _ { \mathsf { A } } { : } { \mathsf { C } } _ { \mathsf { C } } ,$ and relate to the relative stage of DOM decomposition during bioassay studies or naturally occurring degradation gradients.

Of the five identified PARAFAC components, we found the abundance of protein-like $\mathsf { C } _ { \mathrm { T } }$ under initial conditions, to be most closely related to DOM decomposition rates $( k )$ across the six sites (Koehler et al., 2012), which is in agreement with other studies (Guillemette and del Giorgio, 2011). While the intensity of $C _ { \mathrm { T } }$ may have been largely masked in the high DOM lakes due to the dominant abundance of terrestrial humic material (Peaks A and C), we suspect that even the small $\%$ of $\mathsf { C } _ { \mathrm { T } }$ reflects a functionally important fraction of DOM that is of algal and microbial origin, yet is continuously being consumed and replenished, even in a closed system. Moran et al. (2000) did not find a conclusive loss of protein-like Peak T during their incubation study of estuarine DOM. In contrast, another study found that marine-like humic material (Peak M) was produced during an incubation, while protein-like Peak T was consumed (Lønborg et al., 2010). Since DOM fluorescence dynamics from freshwater lakes were predominantly humic-like material ${ < } 8 \% { \mathrm { ~ } } C _ { \mathrm { T } } ,$ with the exception of Ljustjärn $2 0 . 6 \% \mathrm { C _ { T } } .$ ) , it may be challenging to directly compare results with marine studies (Lønborg et al., 2010; Moran et al., 2000) dealing with far lower DOM concentrations $( < 1 \mathrm { m g L } ^ { - 1 } ) $ , and much lower fluorescence of terrestrially derived material. Marine and freshwater DOM have vastly differing levels of degradation history. Thus, there are no clear reasons to expect the dynamics of DOM fluorescence from marine, estuarine and freshwaters to respond similarly during a dark degradation study.

We found that routinely used spectral indexes failed to capture observed change in DOM characteristics during the dark incubation. The minor change in FI during the full incubation period was too small to be considered relevant. Fluorescence intensities used to generate the FI are derived from two emission wavelengths (470:520) at one excitation wavelength $\left( 3 7 0 { \mathrm { n m } } \right)$ positioned on $\mathsf { C } _ { \mathsf { C } }$ (or Peak C). Thus, while we observed an overall reduction in $\mathsf { C } _ { \mathsf { C } }$ fluorescence, the negligible change in FI suggests that the shape and positioning of $\mathsf { C } _ { \mathsf { C } }$ was not greatly influenced during the light-independent DOM processing. The SUVA indicated an increase in aromaticity over time, which is in contrast to other studies reporting a greater loss of color than loss of DOM (Moran et al., 2000; Vähatalo and Wetzel, 2004). The lack of light or photo-bleaching effects during this dark incubation may have reduced the opportunity for selective decomposition of aromatic DOM, resulting in greater biological uptake of nonchromophoric aliphatic DOM, compared to incubations with light. While fluorescence was only measured during the initial and final days of the study, SUVA was monitored more frequently throughout this study, and while we do not present these results here, there was no significant decrease in SUVA at any of the study lakes, even at a higher temporal resolution (Koehler et al., 2012). The change in aromaticity predicted from SUVA is small $( + 3 \% )$ , given the long duration of this dark incubation, and the wide range of initial SUVA, spanning from 1.13 to 3.77. The lack of a significant change in spectral slope is also in contrast to other studies including photodecomposition processes (Galgani et al., 2011; Twardowski and Donaghay, 2002). Surprisingly, the freshness index $\left( \alpha / \beta \right)$ showed no significant change after the 3.5 year dark incubation. Regions of the EEM used to generate the $\alpha / \beta$ include parts of Peak C and Peak M, with Peak C considered to be more decomposed material, and Peak M being fresher DOM (Parlanti et al., 2000). In our study, the overall intensity of both Peaks C and M $\mathrm { C _ { C } }$ and ${ \mathrm { C } } _ { { \mathrm { M } } . }$ ) decreased, which resulted in no consistent directional change in the freshness index. Since the freshness index excludes consideration of changes to Peak A $\left( \mathbf { { \alpha } } \mathbf { { \alpha } } ^ { \prime } \right)$ , which we found to be most resistant to loss processes, the freshness index failed to describe the dark-induced loss of DOM. The overall lack of consistency between the several proxies used to assess changes to aromatic content and extent of decomposition illustrates that these indexes need to be interpreted with care in the case of dark decomposition. Defining differences in the structural composition of $\mathsf { C } _ { \mathsf { A } }$ and $\mathsf { C } _ { \mathsf { C } }$ remains an unanswered question, yet this study provides a clue about the relative reactivity of peaks, since $\mathsf { C } _ { \mathsf { A } }$ was more resistant to decomposition and flocculation in the absence of photo-oxidation.

Other studies have warned of biases due to optical density and high DOM concentrations, which could alter the shape of EEMs, or influence the HIX, yet the identification of a clear threshold level whereby the inner filter correction (with or without dilution) sufficiently accounts for inner filter effects across differing sample types remains undefined (Luciani et al., 2009; Miller et al., 2010; Mobed et al., 1996; Ohno, 2002; Zsolnay, 2002). After testing the influence of inner filter effects across a wide range of concentrations and optical densities from a local humic lake, we demonstrated the reliability of the inner filter correction applied to samples in this incubation study, including the comparison of spectral indexes and $\mathsf { C } _ { \mathsf { A } } { : } \mathsf { C } _ { \mathsf { C } }$ between sites and over time.

# 5. Conclusions

Despite a loss of approximately $4 0 \%$ of the initial DOM during several years of dark incubation, established spectral metrics (SUVA, spectral slope, fluorescence, freshness and humification index) failed to detect consistent changes to DOM degradation. Instead, we found that a simple ratio of the maximum fluorescence intensity of $\mathsf { C } _ { \mathsf { A } } { : } \mathsf { C } _ { \mathsf { C } }$ captured the greatest change in DOM fluorescence characteristics during the incubation. While the chemical and molecular composition of these two peaks remain elusive, the differing responses of these two allochthonous DOM peaks suggest differing reactivity. We found that a greater proportion of $\complement _ { \mathrm { C } }$ is lost relative to $\mathsf { C } _ { \mathsf { A } }$ during dark reactions, irrespective of the initial DOM concentration or prior history of degradation. Changes to the fluorescence spectra observed during this batch study are independent of photolytic reactions or new inputs of DOM, and rather are largely the result of microbial recycling of DOM and flocculation processes. Results from our study also indicate that a preferential loss of Peak C relative to Peak A occurred during the dark incubation. These results have relevance for understanding DOM processing of a sizable portion of the lake water column, since light-induced DOM processing dominates only within the uppermost part of the water column and during summer months in northern boreal regions. The simple index $( \mathsf { C } _ { \mathsf { A } } { : } \mathsf { C } _ { \mathsf { C } } )$ , or Peak A:Peak C, could be a promising proxy to detect dark degradation patterns, and should be evaluated further for its capacity to capture differences in both light and dark dependent degradation states of DOM in natural waters.
