Biogeochemical parameters of pore waters, including dissolved organic matter, nutrients, sulfate, alkalinity, and chloride are explored as convenient and sensitive proxies to monitor the $\mathrm { C O } _ { 2 }$ geological storage sites. Five sites for a $\mathrm { C O } _ { 2 }$ storage project in the Pohang Basin of the East Sea in South Korea were investigated for the preinjection biogeochemical conditions of these sites. Higher dissolved organic carbon $( \sim 3 6 \mathrm { m g L } ^ { - 1 } )$ , chromophoric and fluorescent dissolved organic matter, nutrients, and alkalinity were observed in a fluvially affected acoustic blanking site with geological faults. A general increasing downcore trend of measured DOM parameters, nutrients, and alkalinity with depth was found at the acoustic blanking site affected by riverine runoff with significant correlations among the parameters $( \mathrm { R } ^ { 2 } \colon \sim 0 . 4 - 0 . 8 )$ , highlighting the impact of geological features and external inputs on the downcore biogeochemical properties. The results presented in this study suggest that DOM could be utilized as a robust and complementary biogeochemical parameter.

# 1. Introduction

Carbon capture and storage (CCS) has been suggested to be a solution for anthropogenic $\mathrm { C O } _ { 2 }$ emission that risks the stability of our climate (Holloway and Savage, 1993; Hendriks and Blok, 1995; Bachu and Adams, 2003). CCS procedures typically involve capturing the $\mathrm { C O } _ { 2 }$ usually from major point sources such as power generation plants which accounts for $> 4 0 \%$ world $\mathrm { C O } _ { 2 }$ emission (http://www.iea.org/ co2highlights). And $\mathrm { C O } _ { 2 }$ is then compressed into a supercritical fluid, followed by its injection into deep geological formations, usually $>$ $8 0 0 \mathrm { m }$ , such as saline aquifers or gas/oil fields for long-term storage (Metz et al., 2005). The physical (structural and residual) and geochemical (solubility and mineral) trapping are common trapping mechanisms for $\mathrm { C O } _ { 2 }$ (Szulczewski et al., 2012; Lee et al., 2016). South Korea is one of the largest $\mathrm { C O } _ { 2 }$ emitters in the world, released 657.09 million tons of $\mathrm { C O } _ { 2 }$ in 2013 from fossil fuels consumption (U.S. EIA, 2014). The South Korean government has been exploring potential coastal sites for geological $\mathrm { C O } _ { 2 }$ storage projects.

The Pohang Basin, in the East Sea/Sea of Japan and to the southeast of South Korea, is selected as one of the most promising candidate sites due to favorable geologic structures with thick cap rock and anticline systems, sedimentary strata defined through seismic survey, and its proximity to large industrial complexes (Lee et al., 2016; Hwang et al., 2016). A $1 0 0 \mathrm { t } \mathrm { C O } _ { 2 }$ test injection was performed in January 2017 and further injection in larger scales is planned for 2018 $\left( > 5 0 0 0 \mathrm { t } \right)$ and for 2019 (another $> 5 0 0 0 \mathrm { t } )$ .

Although noble gases and carbon stable isotopes are traditionally used to trace the migration and fate of injected $\mathrm { C O } _ { 2 }$ at geological CO2 storage sites (Gilfillan et al., 2014; Györe et al., 2015, 2017), biogeochemical characteristics such as dissolved organic matter (DOM), nutrients, alkalinity, ${ \mathsf { S O } _ { 4 } } ^ { 2 - }$ , and $\mathbf { C l } ^ { - }$ of pore waters is worthy of investigation as monitoring parameters for tracking the changes in biogeochemical processes because it can readily allow to reflect different characteristics at each site as well as the downcore trends. The organic and inorganic biogeochemical parameters of the non-steadystate pore waters would respond to the massive underground $\mathrm { C O } _ { 2 }$ storage considering the many biogeochemical reactions it is involved, such as $\mathrm { C O } _ { 2 }$ conversion to $\mathrm { H C O } _ { 3 } \mathrm { ^ - }$ , $\mathrm { C O } _ { 2 }$ reduction by $\mathrm { H } _ { 2 }$ to produce $\mathrm { C H } _ { 4 }$ , fixation of $\mathrm { C O } _ { 2 }$ to for example formic acid, methanol (Hong et al., 2013; Van Duc Long et al., 2017). Therefore, storage of $\mathrm { C O } _ { 2 }$ can potentially lead to changes in organic carbon, nutrients, alkalinity, and sulfate. It may also trace potential $\mathrm { C O } _ { 2 }$ leakage and its effects on carbon dynamics because $\mathrm { C O } _ { 2 }$ was found to be reduced to organic carbon in the sediments in proximity of the study sites (Chun et al., 2012), which cannot be seen from the conventional monitoring methods based on noble gases or stable isotopes (Gilfillan et al., 2014; Györe et al., 2015, 2017). It was previously reported that the production of dissolved organic carbon (DOC) and/or chromophoric and fluorescent dissolved organic matter (i.e. CDOM and FDOM) with depth were concurrent with the increases of alkalinity and nutrients $\mathrm { ( N H _ { 4 } ^ { + } }$ and $\mathrm { P O } _ { 4 } { } ^ { 3 - }$ ) and the decrease of sulfate $( { \mathsf { S O } } _ { 4 } { } ^ { 2 - } )$ above the sulfate-methane-transition-zone (SMTZ) in marine sediment pore waters at non-seismic chimney sites (Chen et al., 2016, 2017a). These phenomena have been linked with particulate organic carbon sulfate reduction (POCSR) and anaerobic oxidation of methane (AOM) in oceanic sediments (Burdige and Komada, 2015). At seismic chimney sites, however, the downcore trends could be disturbed by the vertical gas and fluids migration (Chen et al., 2017a). In the consequences, we would expect the effects of geological structure on the depth profile of biogeochemical properties.

In this study, the biogeochemical properties (DOC, CDOM, FDOM, nutrients, alkalinity, and sulfate) of the sediment pore waters at depth ${ \sim } 0 { \ - } 6$ mbsf (i.e. meters below seafloor) were monitored before the substantive $\mathrm { C O } _ { 2 }$ injection (i.e. pre-injection period), primarily utilizing ultraviolet-visible spectroscopy (UV-VIS) and excitation emission matrix coupled with parallel factor analysis (EEM-PARAFAC). The diffusion of pore waters following Fick's First Law of diffusion at the nonchimney sites and the upward migration of basal fluids and gases at seismic chimney sites in the proximity of Pohang Basin have been reported previously (Lasaga and Holland, 1976; Kim et al., 2013). As such, the changes in the depth where $\mathrm { C O } _ { 2 }$ has been stored can potentially be tracked from the shallower sediment pore waters. EEM-PARAFAC presented several advantages over other conventional inorganic monitoring tools, providing complementary biogeochemical information through its fast and easy measurements of optical DOM (i.e. CDOM and FDOM). The aims of this study are three-fold: (1) to compare the biogeochemical characteristics of five $\mathrm { C O } _ { 2 }$ geological storage sites, (2) to explore the downcore trends of biogeochemical features at each site which are usually affected by geological features, and (3) to suggest potential easier-to-measure DOM parameters to complement the conventional inorganic monitoring at the $\mathrm { C O } _ { 2 }$ geological storage sites. In this study, the biogeochemical characteristics and the downcore trends of the pore waters are focused to provide the information in selecting proper environmental monitoring sites for CCS. The post-injection conditions will be investigated in the follow-up studies.

# 2. Materials and methods

# 2.1. Sampling sites

The Pohang Basin is located in the southeastern part of the Korean Peninsula and formed as a pull-apart basin during Miocene back-arc opening of the East Sea/Sea of Japan (e.g., Jolivet et al., 1994; Son et al., 2007). The basin experienced the extensional deformation which developed the fan delta systems at about $1 7 \mathbf { M a }$ and continued until 15 Ma because of the East Sea/Sea of Japan closing resulting from a tectonic inversion (WNW-ESE or NW-SE compression) (Otofuji et al., 1985; Tatsumi et al., 1990).

The Pohang Basin, found in the onshore side as well as in the offshore side, is filled with $> 1$ km thick non-marine to deep-marine sediments, occurring in the eastern part of the NE-SE-trending Yangsan Fault (Yoon, 1975; Chough et al., 1990). The Basin (Yeonil Group) formed along a linkage of contemporaneously developed depositional systems of fan delta, prodelta, slope apron, and basin plain (e.g., Hwang and Chough, 1990). It was reported that the depositional systems in the basin could develop sequentially, separated by several stratigraphic discontinuities and hemipelagic mudstones (Sohn et al., 2001).

The uppermost sediments $( < 7 \mathrm { { m b s f } ) }$ , retrieved from piston cores, consist mostly of muddy sediments, but are partially composed of sandy muds or coarser grains. Chirp profiles are used to pinpoint the location of piston cores (Fig. S1). Piston cores of 16P17 and 16P1a are located at the acoustic turbidity or blanking zone (Fig. S1A and Fig. S1B). This masking suggests the presence of gas-charged sediments. Acoustic facies at the location of 16P03 is characterized by veneered sub-bottom layer overlying well-stratified and channel-fill units (Fig. S1C). An apparent seep-like vertical structure is observed in the channel-fill unit. A relatively thick sub-bottom layer overlies either well-stratified or transparent units at the location of 16P01 (Fig. S1D). The former is laterally terminated against a steep boundary of the latter, suggesting that the boundary could be a fault plane or a basement relief of a sedimentary basin. A piston core of 16P14 penetrates the veneered subbottom layer overlying the transparent to stratified unit (Fig. S1E).

# 2.2. Sampling and analytical techniques

Sediment cores $\yen 70$ length) were collected at five sites (16P01, 16P1a, 16P03, 16P14, and 16P17) by a piston corer in the Yeongil Bay, offshore Pohang Basin, using $\mathsf { R } / \mathsf { V }$ Tamhae II of the Korea Institute of Geoscience and Mineral Resources (KIGAM) in September 2016. Water depths at the five sampling sites were 21, 22, 20, 17, and $2 1 \mathrm { m }$ for 16P01, 16P1a, 16P03, 16P14, and 16P17, respectively (Fig. 1; Table 1). Immediately after the cores were capped and labeled, holes were drilled into the core at ${ \sim } 3 0 { \mathrm { - } } 7 0 \mathrm { c m }$ intervals and the pore water was slowly extracted under ${ \sim } 4 ^ { \circ } \mathrm { C }$ by a Rhizon sampler attached with acidprewashed $2 4 \mathrm { m L }$ -syringe. Extracted pore water was filtered through a $0 . 2 0 \mu \mathrm { m }$ disposable polytetrafluoroethylene filter, before the collection in acid-prewashed high-density polyethylene bottles or glass vials for analyses.

Salinity was determined by a Fisher hand-held portable refractometer immediately after the pore water extraction, calibrated by International Association of Physical Sciences of the Oceans (IAPSO). $\mathbf { C l } ^ { - }$ was analyzed via titration with silver nitrate $\left( \mathsf { A g N O } _ { 3 } \right)$ . Alkalinity was determined by titration with 0.02 M HCl solution. The reproducibility of $\mathrm { { c l } ^ { - } }$ and alkalinity was checked by repeated analysis of IAPSO standard seawater, which was $< 2 \%$ and $< 0 . 5 \%$ , respectively. Sulfate $( \mathsf { S O } _ { 4 } ^ { 2 - } )$ was analyzed by ion chromatography (ICS-1100, Dionex) in the Korea Basic Science Institute (KBSI). $\mathrm { N H _ { 4 } } ^ { + }$ and $\mathrm { P O _ { 4 } } ^ { 3 - }$ were measured spectrophotometrically (Shimadzu UV-2450) at $6 4 0 \mathrm { n m }$ and $8 8 5 \mathrm { n m }$ , respectively (Gieskes et al., 1991; Kim et al., 2013; Chen et al., 2016).

A total organic carbon (TOC) analyzer (Shimadzu TOC-VCPH) was used to measure DOC concentrations as non-purgeable organic carbon. Absorption spectra were scanned from 240 to $8 0 0 \mathrm { n m }$ with a Shimadzu 1800 ultraviolet-visible spectrophotometer. Fluorescence excitation emission matrixes (EEMs) were scanned with a Hitachi F-7000 luminescence spectrometer.

Excitation emission matrix-parallel factor analysis (EEM-PARAFAC) modeling was performed in MATLAB (version 7.0.4) using a DOMFluor toolbox (Stedmon and Bro, 2008). Modeling was validated with splithalf validation $\left( \mathbf { n } = 4 6 \right)$ . An outlier was removed for site 16P17 at depth of $_ { 0 . 6 \mathrm { m } }$ during the modeling. # 2.3. First-order exponential model for DOC CDOM, and FDOM accumulation with depth

A first-order exponential equation (Eq. (1)) (Chen et al., 2017b) was utilized to fit the downcore trends of DOC, $\mathtt { a } _ { 2 5 4 }$ and $\mathbf { a } _ { 3 5 0 }$ , EEM-PARAFAC components, and nutrients and alkalinity (OriginPro8) at a potential non-fault site 16P17:

$$
\mathrm { C _ { d } = a \times e ^ { ( k \times d ) } + b }
$$

where $\mathrm { C _ { d } }$ is the total concentration at depth (d), a and b are constants, e is the natural exponential constant, and $\mathbf { k }$ is the production constant of biogeochemical parameters with depth.

# 3. Results and discussion

# 3.1. Sediment pore water properties and downcore profile of water chemistry

The salinity ranged in average from 33.2 to $3 4 . 0 \mathrm { p s u }$ among five sites (Table 1). The alkalinity of the sediment pore waters generally increased with depth from the top to a depth of ${ \sim } 0 . 6$ mbsf and then remained relatively constant at further depths except for site 16P17 which kept an increasing trend with depth (down to ${ \sim } 2 . 5 \mathrm { m } )$ (Fig. 2). The alkalinity averaged 14.4, 33.9, 12.0, 6.0, and $5 2 . 8 \mathrm { m M }$ for the sites of 16P01, 16P1a, 16P03, 16P14, and 16P17, respectively, with site 16P17 at the highest end while sites 16P14 and 16P03 at the lower end. Nutrients (i.e. $\mathrm { N H _ { 4 } } ^ { + }$ and $\mathrm { P O } _ { 4 } { } ^ { 3 - \cdot }$ ) showed similar trends to that of alkalinity with $\mathrm { N H _ { 4 } } ^ { + }$ concentrations averaged 0.6 to $2 . 2 \mathrm { m M }$ and $\mathrm { P O } _ { 4 } { } ^ { 3 - }$ averaged ${ \sim } 3 5 $ to $2 1 9 \mu \mathrm { M }$ . The concentrations of $\mathrm { N H _ { 4 } } ^ { + }$ generally rose with depth from the top to a depth of ${ \sim } 0 . 6$ mbsf and then stayed relatively constant at further depths except for site 16P17, which showed a constant increasing trend. For $\mathrm { P O } _ { 4 } { } ^ { 3 - }$ , however, the downcore profile was slightly different from that of $\mathrm { N H _ { 4 } } ^ { + }$ , with site 16P17 remaining relatively constant from ${ \sim } 1 . 6$ mbsf to the lower depths.

Similar downcore accumulation trends between the nutrients of $\mathrm { N H _ { 4 } } ^ { + }$ and $\mathrm { P O _ { 4 } } ^ { 3 - }$ was observed in Arctic and East Sea marine sediments where less affected by terrestrial inputs (Chen et al., 2016, 2017a). In freshwater riverine sediments, however, the opposite downcore trend of $\mathrm { P O } _ { 4 } { } ^ { 3 - }$ was found to those in marine sediments (Chen et al., 2017b). The different trends observed between the two contrasting aquatic systems can be attributed to the capture of $\mathrm { P O _ { 4 } } ^ { 3 - }$ by iron compounds in iron-rich anaerobic freshwater sediments as opposite to the release of $\mathrm { P O } _ { 4 } { } ^ { 3 - }$ during microbial reduction in marine sediments, in which Fe compounds are converted to iron-sulfides (Roden and Edmonds, 1997). In the present study, the Pohang Basin, a marine environment, is affected by the Hyeong River inputs. This geographical location and the subsequent effect of terrestrial input might be partially responsible for the unique downcore trend of $\mathrm { P O } _ { 4 } { } ^ { 3 - }$ which generally increased at top sediments like in the marine systems (Chen et al., 2017a), but dipped as in the freshwater systems (Chen et al., 2017b), and then largely remained constant further downward (Fig. 2). These trends were in line with the condition of the studied region with the influences of riverine inputs in the shallow (water depth $< 2 2 \mathrm { m } )$ ) coastal environment. In contrast, ${ \ S O _ { 4 } } ^ { 2 - }$ exhibited completely different profiles with depth from the inorganic parameters mentioned above. For example, the ${ \ S O _ { 4 } } ^ { 2 - }$ concentrations sharply decreased from ${ \sim } 2 7 . 3 \mathrm { m M }$ to ${ \sim } 0 \mathrm { m M }$ at depth of 1.1 mbsf at sites 16P17 and 16P1a, while they decreased at site 16P01 from ${ \sim } 2 8 \mathrm { m M }$ to ${ \sim } 2 0 \mathrm { m M }$ at depth of 0.6 mbsf and kept relatively constant downwards. The nutrients, alkalinity, and sulfate levels are generally comparable to those in other sediment pore waters, such as in the coastal Ulleung Basin of the East Sea, the Arctic shelf, and major South Korean rivers (Table S1).

The specific absorption coefficient $\mathbf { a } ^ { * } { } _ { 2 5 4 }$ averaged between 4.0 and 7.5, falling in the range of those observed in the pore waters at coastal Yangtze Estuary (Wang et al., 2013, 2014). The BIX averaged ${ \sim } 0 . 7 \substack { - 0 . 8 }$ , similar to those observed in the Ulleung Basin of the East Sea (Fig. 1A, Chen et al., 2017a). The HIX averaged 2.5–3.8, comparable to those reported in the coastal Yangtze Estuary (Wang et al., 2014) but lower than those found in deeper sediment (192–238 mbsf) pore waters in the Ulleung Basin of the East Sea/Sea of Japan (HIX: 9.5–9.7, Chen et al., 2017a).

EEM-PARAFAC modeling identified three independent components: $C _ { \mathrm { ~ < ~ } 2 6 0 / 4 3 4 }$ , C320/422, and $\mathrm { C } _ { 2 8 0 / 3 0 2 }$ $( \mathsf C _ { \mathrm { ~ < ~ } 2 6 0 / 4 3 4 }$ stands for a fluorescent component with excitation and emission maxima at $< 2 6 0 \mathrm { n m }$ and $4 3 4 \mathrm { n m }$ , respectively, Fig. 4) and the spit-half validation is shown in the supporting information (Fig. S3). According to the spectra, $C _ { \mathrm { ~ < ~ } 2 6 0 / 4 3 4 }$ , $\mathrm { C } _ { 3 2 0 / 4 2 2 }$ , and $\mathrm { C } _ { 2 8 0 / 3 0 2 }$ are traditionally termed as terrestrial humic-like, marine humic-like, and protein-like components, respectively, although they could be sourced from both terrestrial and marine ecosystems. It is notable that polyphenols, which are mostly derived from terrestrial organic sources, may display similar spectral position to proteinaceous materials (Maie et al., 2007). Similar to previous CDOM parameters, the FDOM at site 16P17 was one order of magnitude higher than at the remainder sites (Table 1, Fig. 4). Terrestrial humic-like C1 was $2 7 . 9 ~ \pm ~ 1 4 . 4$ RU at site 16P17, as compared to $2 . 2 \ \pm \ 0 . 8$ RU, $5 . 7 ~ \pm ~ 2 . 5 \mathrm { R U }$ , $2 . 3 ~ \pm ~ 1 . 0$ RU, and $1 . 8 ~ \pm ~ 0 . 8$ RU at sites 16P01, 16P1a, 16P03, and 16P14, respectively. Likewise, humic-like C2 at site 16P17 was $1 1 . 0 ~ \pm ~ 4 . 8 \mathrm { R U }$ , topped among the five sites with the rest averaged $1 . 6 ~ \pm ~ 0 . 5 ~ \mathrm { R U }$ , $3 . 3 ~ \pm ~ 1 . 3$ RU, $1 . 5 ~ \pm ~ 0 . 5$ RU, and $1 . 2 \pm \ : 0 . 5$ RU. For protein-like C3, the intensity site was $1 0 . 0 ~ \pm ~ 3 . 9 ~ \mathrm { R U }$ at site 16P17 whereas the others averaged $3 . 1 ~ \pm ~ 1 . 0$ RU, $3 . 7 ~ \pm ~ 1 . 1$ RU, $3 . 3 ~ \pm ~ 0 . 7$ RU, and $2 . 7 \ \pm \ 0 . 4$ RU. The relative abundance of C1 (i.e. terrestrial humic-like component) was the highest at 16P17 among the five sampling sites (Table 1), suggesting that the site is the most likely influenced by terrestrial organic sources compared to other sites.

# 3.3. Downcore profiles of DOM parameters at site 16P17

The DOM parameters at fluvially affected acoustic blanking site 16P17 showed a general increasing trend with depth while no clear trends were observed for the remainder sites (Fig. 3, Fig. 4). These trends are generally in line with previous findings in the coastal ecosystems (Burdige et al., 2004; Wang et al., 2013; Chen et al., 2016, 2017a), as well as in freshwater sediments (Chin et al., 1998; O'Loughlin and Chin, 2004; Ziegelgruber et al., 2013; Chen et al., 2017b). There is an outlier at site 16P17 at a depth of ${ \sim } 0 . 6$ mbsf, implying a potential disturbance and complicated geological properties in the studied area. The DOC increased from $1 0 . 9 \mathrm { m g L } ^ { - 1 }$ at a depth of 0.1 mbsf to $5 3 . 0 \mathrm { m g L } ^ { - 1 }$ at a depth of 5.6 mbsf at site 16P17. The absorption coefficient increased from $3 3 . 7 \mathrm { m } ^ { - 1 }$ to $2 6 6 . 7 \mathrm { m } ^ { - 1 }$ for $\mathtt { a } _ { 2 5 4 }$ and from $9 . 1 \mathrm { m } ^ { - 1 }$ to $5 3 . 5 \mathrm { m } ^ { - 1 }$ for $\mathtt { a } _ { 3 5 0 }$ . As for FDOM, the levels increased from 0.5 RU to 52.3 RU for $C _ { < 2 6 0 / 4 3 4 }$ , from 0.6 RU to 17.4 RU for $\mathrm { C } _ { 3 2 0 / }$ 422, and $2 . 8 ~ \mathrm { R U }$ to 16.8 RU for $\mathrm { C _ { 2 8 0 / 3 0 2 } }$ . As geological survey implied the site 16P17 is a relative stable acoustic blanking site (Fig. S1A), further discussion will continue focusing on this site. Considering its trends of biogeochemical parameters with depth consistent with previous reports (Chen et al., 2016; Chen et al., 2017b), this site appears to be suitable for a long-term monitoring of a DOM profile as a way of comparing biogeochemical conditions before and after $\mathrm { C O } _ { 2 }$ injection.

3.4. Accumulation of DOC and optical DOM concurrent with increase of nutrients and alkalinity at fluvially affected acoustic blanking site 16P17

First order accumulation of DOC, CDOM, and FDOM $( \mathrm { R } ^ { 2 } = 0 . 5 6$ to 0.67, $p \ : < \ : 0 . 0 5 )$ and nutrients and alkalinity $( \mathrm { R } ^ { 2 } = 0 . 7 7$ to 0.78, $p \ < \ 0 . 0 1 \}$ with depth was observed for the pore waters at site 16P17 with the outlier at a depth of $_ { 0 . 6 \mathrm { m } }$ excluded (Fig. 5). The rate constant (k) ranged from 0.0023 to $0 . 0 6 \mathrm { m } ^ { - 1 }$ for DOM parameters, and it was $0 . 0 7 \mathrm { m } ^ { - 1 }$ and $0 . 2 8 \mathrm { m } ^ { - 1 }$ for alkalinity and $\mathrm { N H _ { 4 } } ^ { + }$ , respectively (Table S1). In contrast, ${ \mathsf { S O } _ { 4 } } ^ { 2 - }$ decreased sharply with a constant rate of $3 . 0 7 \mathrm { m } ^ { - 1 }$ . The accumulation rate constants for DOC and absorption coefficients (i.e. $\mathtt { a } _ { 2 5 4 }$ and $\mathbf { a } _ { 3 5 0 , }$ ), however, are generally 1–2 orders of magnitude lower than those in another basin (Ulleung Basin) in the East Sea/Sea of Japan $( 0 . 0 0 3 - 0 . 0 2 \mathrm { m } ^ { - 1 }$ vs. $0 . 1 0 { - } 0 . 1 7 \mathrm { m } ^ { - 1 } ;$ ), despite its relatively higher concentrations of DOC $1 0 . 9 \mathrm { m g L } ^ { - 1 }$ vs. $5 . 5 \mathrm { m g L } ^ { - 1 } )$ ) and CDOM absorption coefficients $\mathtt { ( a _ { 2 5 4 } }$ : $3 3 . 7 \mathrm { m } ^ { - 1 }$ vs. $1 . 9 \mathrm { m } ^ { - 1 }$ ; a350: $9 . 1 \mathrm { m } ^ { - 1 }$ vs. $0 . 9 \mathrm { m } ^ { - 1 }$ ) at surface sediments, suggesting high variability of accumulation rates at geographically different sites. In addition, the accumulation rate constant for DOC here corresponds to the lower end of those observed in the pore waters from South Korean river sediments affected by weir-impoundment. This low accumulation might be related to the fast consumption of sulfate at site 16P17, where the sulfate concentration reached almost zero at a depth of $\mathrm { \sim } 1 . 1$ mbsf, contrasting with a relatively deeper sulfate-methane-transition-zone depth (SMTZ: ${ \sim } 4 . 1$ mbsf) at a non-chimney U1 site in the Ulleung Basin of the East Sea/Sea of Japan (Fig. 1A, Chen et al., 2017a). It was previously reported that DOM net accumulation was positively correlated with particulate organic carbon sulfate reduction (POCSR) rates in sediments pore waters (Chen et al., 2016). Thus, the depletion of sulfate in our studied site may explain the low production rate of DOM.

The accumulation trend of DOM with depth explain the observation of the exceptionally higher DOC and optical DOM at acoustic blanking site 16P17 compared to other sites, including sites potentially at/along faults and an acoustic blanking site 16P1a which is farther away from the river mouth. Site 16P17 is most likely to be associated with the relatively undisturbed geological structure as it is an acoustic blanking site (Fig. S1A) and meanwhile close to terrestrial inputs (Fig. 1B). In contrast, the other sites at geological faults, i.e. 16P01, 16P03, and 16P14, which might be affected by geological faults, might experience the migration of gas and fluids to greater extents without the occurrence of DOM accumulation with depth. In addition, another acoustic blanking site farther away from the riverine runoff showed relatively subtle accumulation of DOM to a depth $\mathrm { \sim } 1 . 1$ mbsf, potentially due to the usually more biolabile marine DOM sources at this site. Good positive correlations between DOC and nutrients or alkalinity were primarily found for fluvially affected acoustic blanking site 16P17 $( \mathrm { R } ^ { 2 } = 0 . 8$ , 0.5, 0.8 for $\mathrm { N H _ { 4 } } ^ { + }$ , $\mathrm { P O } _ { 4 } { } ^ { 3 - }$ , and alkalinity, respectively, $\mathbf { n } = 1 0$ , $p \ < \ 0 . 0 5$ , Table S2, Fig. S2). Meanwhile, a linear inverse correlation was observed between DOC and ${ \ S O _ { 4 } } ^ { 2 - }$ at site 16P17 $( { \bf r } = 0 . 7 0$ , ${ \mathfrak { n } } = 1 0 , p \ < \ 0 . 5$ , Table S2). Significant correlations between DOC and $\mathrm { N H _ { 4 } } ^ { + }$ were also found at sites 16P1a and 16P14, which are relatively farther away from the possible fault than sites 16P01 and 16P03 (Fig. S1A).

# 3.5. Potential application of DOM parameters for biogeochemical monitoring at $C O _ { 2 }$ geological storage sites

There were consistent positive linear correlations between DOM parameters (DOC, $\mathtt { a } _ { 2 5 4 }$ and $\mathbf { a } _ { 3 5 0 }$ , and EEM-PARAFAC components) and nutrients $\mathrm { ( N H _ { 4 } } ^ { + }$ and $\mathrm { P O } _ { 4 } ^ { 3 - ^ { . } }$ ) and alkalinity, while inverse correlations between DOM parameters and sulfate $( \mathsf { S O } _ { 4 } ^ { 2 - } )$ at site 16P17 as well as for all samples (Table 2, Fig. S3). The correlation coefficients ranged from 0.60 up to 0.93 $( p < 0 . 0 5 )$ for all the relationships. The close associations between the two monitoring parameters suggest the existence of certain biogeochemical processes coupling between DOM and inorganic constituents in the sediment pore water. Furthermore, these

DOC and DOM optical parameters can be measured relatively fast with a high sensitivity without solvents, making it favorable to monitor DOM at $\mathrm { C O } _ { 2 }$ geological storage sites in near-real time. In addition, based on the relationships among the biogeochemical parameters, the nutrients, alkalinity, and sulfate could be inferred from the DOM proxies. This additional monitoring can also provide further information on organic carbon dynamics at these sites as $\mathrm { C O } _ { 2 }$ conversion to $\mathrm { C H } _ { 4 }$ as well as enzymatic carbon fixation into other non-purgeable organic matter in the sediments is possible (Chun et al., 2012; Hong et al., 2013; Van Duc Long et al., 2017), which is not feasible by the conventional measurements of inorganic parameters alone.

# 4. Conclusions

Different characteristics and downcore trends of geochemical parameters were observed between a fluvially affected acoustic blanking site and possible fault sites. A general first-order accumulation trend of DOM and nutrients with depth was found at the acoustic blanking site affected by riverine runoff. The strong correlations between DOM parameters and nutrients and/or alkalinity make it possible to conveniently and rapidly monitor DOM as additional biogeochemical parameters at $\mathrm { C O } _ { 2 }$ geological storage sites. The optical measurements of DOM could be utilized as robust and complementary tools to conventional inorganic geochemical parameters for monitoring purpose. Furthermore, the DOM parameters could provide critical information with respect to the carbon dynamics which is closely involved with $\mathrm { C O } _ { 2 }$ change as inorganic carbon can be converted into organic forms via various biological pathways. The rapidness and high sensitivity of spectroscopic measurements for pore water DOM can be viewed as a promising option to monitor the biogeochemical characteristics of $\mathrm { C O } _ { 2 }$ geological storage sites in the future.
